        第七章   为病人举行礼仪时的任选经文

 

壹、圣经选读

 

下列选读可用于病人弥撒、病人傅油礼仪、或慰问病人时；也可用于为病人举行的圣经诵祷中。读经的选择当以牧灵因素，以及病人的神形状况为标准。有些读经是更适合于垂危病人的。

 

一、旧约读经

列上 19 : 1 - 8 ( 厄里亚先知在路上疲惫不堪，上主遣天使，给他送食物 )  那时候，阿哈布将厄里亚所作的一切，全告诉了依则贝尔……

 

约 3 : 1 - 3, 11 - 17, 20 - 23 ( 为何将光明赐给不幸者？)  那时候，约伯开口说……

 

约 7 : 1 - 4, 6 - 11 ( 记住！我的生命像似一口气。)  那时候，约伯说：人生像是服兵役……

 

约 7 : 12 - 21 ( 人算什么，你竟如此显扬他？)  那时候，约伯说：我岂是海洋……

 

约 19 : 23 - 27 ( 我确实知道为我伸冤者还活着 ) 那时候，约伯说：谁能帮助我，把我的话都记下来……

 

智 9 : 9 - 11, 13 - 18 ( 你若不赐予智慧，谁能知道你的旨意？) 智慧与你同在，他洞悉你的工作……

 

依 35 : 1 - 10 ( 你们应加强萎弱之手) 荒野和不毛之地必要欢乐……

 

依 52 : 13 - 53 : 12 ( 他承担了我们的罪过 ) 请看我的仆人必要成功……

 

依 61 : 1 - 3a ( 上主的神派遣我安慰破碎的心灵 )上主的神临于我身上……

三、答唱咏

依 38 : 10, 12, 16

答：上主！是你保存了我的生命脱免灭亡的深渊。

一、我曾说过，我正值中年，就要去世；我的余年将奔向阴府之门。( 答 )

二、我的居所被拆去，好像牧童的帐幕被拔除；你好像织工卷起我的生命，由纺织机上把它割断，你从早到晚总想结果我。( 答 )

三、吾主，我的心灵唯有仰赖你！求你使我的精神复苏，使我复原，使我生存。( 答 )

 

咏 6 : 2 - 4, 2, 4b-6, 9 - 10

答：现在我心烦乱，父啊！救我脱离这时刻吧！

一、上主，求你不要含怒责罚我，不要在气愤中惩戒我。上主，我的体力衰弱，求你怜恤我；上主，我的骨骼战栗，求你医治我。我的心神惊惶已极，上主，你要等到何时？ ( 答 )

二、上主，求你回来援助我，你是慈悲的，求你拯救我。在死人中，谁会想起你？在阴府里，谁来称谢你？( 答 )

三、上主俯允了我的哀号，上主接受了我的祈祷。我的仇敌必要惭愧惊惶，转瞬之间，含羞逃亡。( 答 )

 

咏 24 : 4 - 9 , 10, 14, 15 - 16

答 : 我的心神仰望你。

一、上主，请指示我的道路，教给我你的途径。求你教训我，以真理指导我，我终日仰望你，因为你是我的救主。( 答 )

二、上主，求你念及你的仁爱，你一向就以慈悲为怀。上主，求你忘掉我青春的罪愆与过犯；你慈爱宽厚，请垂念我。( 答 )

三、 因为上主温良而公正，指示罪人应走的道路；引导谦卑的人履行正义，教训善良的人遵循坦途。 ( 答 )

四、上主对遵守他盟约和诫命的人，表现他的慈爱和真诚。上主亲切对待敬畏他的民众，使他们了解他的盟约。( 答 )

五、我仰望上主，目不转睛，因为他使我脱离陷阱，我孤苦伶仃，求你回顾我，怜悯我。 ( 答 )

 

咏 26 : 1, 4, 5, 7 - 10

答：你要鼓起勇气，振作精神，期望上主。

一、上主是我的光明，我的救援，我还畏惧何人？上主是我生命的保障，我尚害伯何人？( 答 )

二、我只向上主请求一事：愿我一生一世住在上主的殿里。凝视上主的美善，参拜上主的圣殿。( 答 )

三、因为在我困苦的时日，他把我藏在他的帐幕中，把我藏在他帐幕的深处，把我高置在磐石上。( 答 )

四、上主，求你俯听我的呼声，求你怜悯我，俯允我。上主，我切望仰瞻你的慈颜。求你不要转面不顾。( 答 )

五、上主，我在寻求你的慈颜。求你不要转面不顾，你发怒时，不要赶走你的仆人。( 答 )

六、你一向是我的救援，我的救主，不要把我遗忘。我的父母虽然拋弃了我，但是上主却收容了我。( 答 )

 

咏三三， 2 - 7, 17, 19

答：请你们体验并观看，上主是何等美善！

一、我要时时赞美上主，我的口舌不断赞颂上主。我的心灵因上主而自豪，谦卑的人听了也必欢跃。( 答 )

二、请你们同我一起赞扬上主，让我们齐声颂扬他的名号。我寻求了上主，他就应允了我，他救我脱离了一切恐惧。( 答 )

三、你们仰瞻他，就会喜形于色，绝不致满面羞惭。卑微的人一呼求，上主立刻俯允，拯救他脱离一切艰辛。( 答 )

四、上主的威容敌视作恶的人民，把他们的记念从人间抹去。上主亲近破碎的心灵，医治人精神的苦痛。( 答 )

 

咏 41 : 2 - 3 ; 42 : 3 - 4

答：上主，我的心渴慕你，就像小鹿渴望清泉。

一、我的心灵渴慕天主、生活的天主，我何时才能见到天主的慈颜？( 答 )

二、求你赐下光明和真理，引我到你的圣山，进入你的殿宇。( 答 )

三、我要走向上主的祭坛，到使我心喜的天主台前；天主，我的天主，我要弹琴赞颂你。 ( 答 )

 

咏 62 : 2 - 6 , 8 - 9

答：上主，我的天主，我的灵魂渴慕你。

一、天主，你是我的天主，我急切寻觅你；我的灵魂渴慕你，我的肉身切望你，好像一块干涸无水的田地。昔日我曾在圣殿里瞻仰过你，为看到你的威能，和你的光辉。( 答 )

二、 因为你的慈爱比生命更为宝贵，我的唇舌称扬赞颂你。我要一生一世这样赞美你，我要奉你的名把双手举起。我的灵魂好像饱飨了膏脂，我要以欢愉的口唇赞美你。( 答 )

三、我在床上想起了你，彻夜不眠地思念着你。我的心神紧紧地追随你，你的右手时时扶持我。( 答 )

 

咏 70 : 1 , 2, 5, 6, 8, 9, 14, 15

答：我的上主，请快来救我。

一、上主，我投靠你，永远不会蒙受羞辱！你是公正的，求你救拔我，求你侧耳俯听我，速来拯救我。( 答 )

二、因为，上主，你是我的寄望，从我幼年时，你就是我唯一的仰仗；我尚在母胎中，你就保护我，我出世以来，就仰赖你。( 答 )

三、我要满口赞美你，我要终日颂扬你。我年老时，求你不要把我拋弃；我力衰时，求你不要与我远离。( 答 )

四、至于我，我仍然不断地期望，对你更加不断地赞扬。我的口要传述你的宽仁，终日宣扬你无数的救恩。( 答 )

 

咏 85 : 1 - 2, 3 - 4, 5 - 6, 11, 12 - 13, 15 - 16

答：上主，求你俯听我的祈祷。

一、上主，求你侧耳俯听我，因为我困苦贫穷。求你保护我的灵魂，因为我对你虔城；求你拯救那仰赖你的仆人。( 答 )

二、我的天主，求你怜悯我；上主，我整天呼求你。求你使你的仆人欢喜，上主，我的心灵仰赖你。( 答 )

三、上主，你实在良善宽仁，以仁慈厚待呼求你的人。上主，求你俯听我的祈祷，细听我的哀告。( 答 )

四、我的天主，我要全心向你歌颂，上主，我要永远光荣你的圣名。因为你对我的仁慈，伟大无边，拯救我的灵魂，免陷阴间。( 答 )

五、上主，你却是良善慈悲的天主，极其宽仁而信实，不轻易发怒。求你垂视我、怜悯我，赐给你的仆人活力。( 答 )

 

咏 89 : 2 - 6, 10b, 12, 14, 16

答：上主，你从永远就是我的依靠。

一、群山尚未诞生，宇宙尚未出现，你就是无始无终的天主。( 答 )

二、你命世人归于灰土说：「归来吧！亚当的子女！」因为一千年在你眼前，就好像刚过去的昨天，又像夜里的一更时间。( 答 )

三、你消灭他们有如清晨一觉，他们就好像刚出生的青草。早晨还花开繁茂，晚上就萎谢枯槁。( 答 )

四、人生时光大半是劳苦和虚空，转瞬即逝，我们也飘落无踪。求你教我数算自己的年岁，使我学得内心的智慧。( 答 )

五、使我们清晨就饱飨你的慈爱，欢欣鼓舞，天天愉快。求你使你的仆人得见你的功业，向他们的子孙彰显你的荣耀。( 答 )

 

咏 101 : 2 - 3 , 24 - 25, 26 - 28, 19 - 21

答：上主，求你俯听我们的祈祷。

一、我遭难之日，求你不要转面不顾我；我呼求你时，求你快快俯听我！因为我的岁月有如过眼云姻，我的骨头枯焦有如灰炭。( 答 )

二、我说：「我的天主！」你的生命无始无终，不要让我中年丧生。你自太初奠定了大地尘寰，亲手造成了诸天。( 答 )

三、天地终将毁灭，你却永恒不变，万物都要像衣服一样破烂。你变化万物有如更换衣裳，你始终如一，万寿无疆。( 答 )

四、要将此事记录传于后世，使新生的子民赞美上主。上主从至高的圣所俯瞰，由高天之上垂顾人间 : 倾听俘虏的哀歌，释放待毙的囚犯。( 答 )

 

咏 102 : 1 - 4, 11 - 18

答：上主富于仁爱宽恕。

一、我的心灵，请歌颂上主，我的五内，请赞美他的圣名。我的心灵，请歌颂上主，不要忘记他的一切恩宠。( 答 )

二、他赦免了你所有的罪行，治愈了你的各种病症；他救你脱离阴间，给你加上仁爱和慈悲的荣冠。( 答 )

三、天离地有多么高，他的慈爱对敬畏他的人也多么深厚；东离西有多么远，他使罪过离开我们也多么遥远 ( 答 )

四、父亲怜爱儿女，同样，上主也怜爱敬畏他的人。他知道我们怎样形成，也记得我们不过是灰尘。( 答 )

五、世人的岁月有如青草丛生，又像田野的花卉茂盛。东风一吹过，就不复存在，连它生长之处也把它忘怀。( 答 )

六、上主永远怜爱敬畏他的人，他的正义也惠及他们的子孙。他们遵守他的誓盟，忠实履行你的诫命。( 答 )

 

咏 122 : 1 - 2

答：我的眼目仰望上主，期望他怜悯我们。

一、高居诸天的上主，我举目仰望你。正如仆人的眼睛望着主人的手。( 答 )

二、又像婢女的眼睛注视主母的手；我们的双目也同样仰望着上主、我们的天主，直到他怜悯我们为止。( 答 )

 

咏 142 : 1 - 2, 5 - 6, 10

答：上主，求你俯听我的祈祷。

一、上主，求你俯听我的祈祷，侧耳聆听我的哀号；凭你的真诚和正义俯允我！请不要审讯你的仆人，因为众生在你面前不得称为义人。( 答 )

二、我回忆以往的时日，迫念你的一切奇事，凝思你手中的作为。我不禁向你伸开双手，我心渴望你，有如旱田之望甘雨。( 答 )

三、你是我的天主，求你教导我奉行你的旨意。愿你的善神引导我，走上坦途。( 答 )

 (1) 为年老者

上主！你的仆人年老患病；为了拯救他的灵魂肉身，求你藉这傅油，恩赐宠佑，惠然眷顾，使他由圣神获得充沛的力量，能保持坚强的信德和望德，给众人树立忍耐的榜样，并对你的慈爱照顾，表示出衷心的喜乐和感激。以上所求是靠我们的主基督。( 答：阿们 )  

 

(2) 为垂危病人

主耶稣基督，人类的救主！你受尽苦难，承担了我们的痛苦和脆弱。现在，我们为这位患病的弟兄／姊妹 ( 某 )。谦恭地求你，使他获得你的救赎，坚定他得救的信念，减轻他身心的痛苦。你是天主，永生永王。( 答：阿们 )

 

(3) 为领傅油及临终圣体的病人

天主，仁慈的圣父！你是受苦人的安慰，求你惠然眷顾这位依恃你的仆人 ( 某 )，使他的病苦，因领受傅油的恩宠而获得减轻，并使他所领的圣体 ( 圣血 )，成为他旅途上的食粮，帮助他迈向永生。以上所求是因我们的主基督。( 答 : 阿们 )

 

(4) 为弥留的病人

仁慈的圣父！你认识心地善良的人，乐于宽赦人的罪过，你从不拒绝向你求饶的人；求你垂怜这位弥留中的仆人 ( 某 )，使他藉傅油圣事及我们发自信德的祈祷的扶助，减轻他身心的痛苦，使他获得罪赦，并因你的慈爱获得力量。因你的圣子基督，为我们战胜了死亡，开启了永生之门，他与你永生永王。( 答 ：阿们 )
