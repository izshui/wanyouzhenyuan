第三章  病人傅油弥撒

 

应注意事项 

(一) 如病人的状况许可，尤其当病人要领圣体时，则病人傅油可在弥撒中举行，此弥撒可在圣堂中，或经教会教长同意，也可在病人家中，或在医院中合适的地方举行。

(二) 如病人傅油在弥撒中举行，神父应穿白色祭衣，举行此「病人弥撒」。但如遇到将临期、四旬期及复活期主日、节日、圣灰礼仪日或圣周时，则应单行当天的弥撒，但可用特别为病人的「最后祝福式」。

(三) 读经可由为病人的任选弥撒读经中，或由本礼典第七章中选取。除非为了病人及参礼者的益处而另行选择。

礼规禁止举行「病人弥撒」时，在当天弥撒中可由以上所提供的读经中选读一篇；除非遇到圣周三日庆典、耶稣圣诞、主显节、耶稣升天、圣神降临、天主圣三、耶稣圣体圣血、以及其他法定的节日。

(三) 傅油礼在福音及讲道后，照以下顺序举行：

(1) 福音后，神父以经文为出发点讲解疾病在救恩史中的意义，以及傅油圣事的恩宠，但须顾及病人的状况和环境。

(2) 傅油礼由「祷文」开始，如此祷文在傅油后念，或傅油后念「信友祷词」时，则由「覆手礼」开始。然后，如无主教祝圣的圣油，则祝圣病人油；有祝圣过的病人油时，则念「感谢词」，随后，即行傅油。

(3) 傅油礼后念「信友祷词」，除非傅油礼前已念过「祷文」。此信友祷词以傅油礼后的综合祷词结束。然后弥撒照常规继续进行，即继续行准备饼酒礼。病人及参礼者可兼领圣体圣血。

二、圣道礼仪 ( 坐下 )

7. 读经一 ( 他确实背负了我们的疾苦 )

恭读依撒意亚先知书 ( 53 : 1-5, 10-11 )

有谁会相信我们的报道呢？上主的手臂又向谁显示了呢？他在上主面前生长如同嫩芽，又像出自干地中的根苗；他不俊美，也没有威仪，可使我们瞻仰；他的仪表平凡，不能使我们恋慕。他受尽了侮辱，被人遗弃；他真是个苦人，熟悉病苦；他好像一个人们不屑一顾的人，他受尽了侮辱，因而我们都以为他不算什么。然而他承受了我们的疾苦；担负了我们的疼痛；我们还以为他受了惩罚，是天主所击伤，和受贬抑的人。可是他被刺透，是因为我们的悖逆；他被打伤，是因为我们的罪恶，因为他受了责罚，我们便得了安全；因为他受了创伤，我们才得以痊愈。上主的旨意是要用病苦折磨他； 当他牺牲了自己的性命，作了赎罪祭时，他要看见他的后辈延年益寿，上主的旨意也藉他的手得以实现。在他受尽了痛苦之后，要看见光明，并因自己的经历而满足；我公义的仆人要使多人成为义人，因为他承当了他们的罪过。--- 以上是天主的圣训。

信友：感谢天主。

侍友：主，愿光荣归于你。

那时候，耶稣来到伯铎家里，看见他的岳母发烧，躺在床上，耶稣一摸她的手，热症就退了。她便起来伺候耶稣。到了晚上，有些人给耶稣送来了许多附魔的人，他只用一句话就驱逐了恶神；治好了所有的病人。这样，就应验了依撤意亚先知所说的话:「他承受了我们的脆弱，担负了我们的疾病 」--- 以上是天主的圣训。

信友：基督，我们赞美你。

12. 信经( 如按礼规当念 ) ( 起立 )

全体：我信唯一的天主，全能的圣父，天地万物，无论有形无形，都是他所创造的。我信唯一的主、耶稣基督、天主的独生子。他在万世之前，由圣父所生。他是出自天主的天主，出自光明的光明，出自真天主的真天主。他是圣父所生，而非圣父所造，与圣父同性同体，万物是藉着他而造成的。他为了我们人类，并为了我们的得救，从天降下。( 鞠躬 ) 「他因圣神由童贞玛利亚取得肉躯，而成为人。」他在般雀比拉多执政时，为我们被钉在十字架上，受难而被埋葬。他正如圣经所载，第三日复活了。他升了天，坐在圣父的右边。他还要光荣地降来，审判生者死者，他的神国万世无疆。我信圣神。他是主及赋予生命者，由圣父圣子所共发。他和圣父圣子，同受钦崇，同享光荣，他曾藉先知们发言。我信唯一、至圣、至公、从宗徒传下来的教会。我承认赦罪的圣洗，只有一个。我期待死人的复活，及来世的生命。阿们。

 

病人傅油 

1. 祷文

主祭：亲爱的弟兄妹妹们：请大家怀着坚强的信德，为我们的弟兄／姊妹 ( 某 )，向主耶稣恳切祈祷：

---求主仁慈地看顾他，并以圣油坚强他。答 : 求主俯听我们。

---求主拯救他脱免一切凶恶。答：求主俯听我们。

---求主减轻病人的痛苦。答：求主俯听我们。

---求主助佑照顾病人的视友。答：求主俯听我们。

---求主拯救他脱离罪恶及一切诱惑。答：求主俯听我们。

---我们因主的名在病人头上覆手，求主赐给他活力与健康。答：求主俯听我们。

 

2. 覆手  

( 神父在病人头上覆手，不念经文。覆手表示祈求天主赐予圣神和神的恩宠：医治和降福病人 )

 

3. 祝圣病人油  

( 如果没有经主教祝圣过的油，则主礼神父可用以下经文祝圣 )

请大家祈祷 : 天主，予人安慰的慈父！你曾藉你的圣子医治人的疾苦；求你俯听我们出自信心的祈祷：从天上派遣你施慰的圣神，降福这取自植物的油脂，以滋养我们的身体；愿您的降福使所有敷用这油的人，都得身体和心灵的保障，除去一切痛苦、衰弱和疾病。

天主！求你使这经你圣化的油，能为我们产生治病的神效。因我们的主基督之名。

答 : 阿们。

 

( 如果用已经祝圣过的油，则念以下感恩祷用：)

领：全能的天主圣父！我们赞美你，你为了我们，并为了我们的得救，派遣你惟一圣子来到世上。( 答：赞美天主 )

领：天主的惟一圣子！我们赞美你，你自愿降临人间，医治我们的病苦。(答：赞美天主 )

领 : 施慰的天主圣神！我们赞美你，你以永恒的德能，坚强我们懦弱的肉躯。 (答：赞美天主 )

领：上主！求你使敷用这圣油的信徒，能减轻他的痛苦，治愈他的疾病。以上所求是靠我们的主基督。( 答：阿们 )

 

4. 傅油礼  

( 神父取圣油傅抹病人的额和双手，同时念一次下列经文：)

藉此神圣的傅油，并赖天主的无限仁慈，愿天主以圣神的恩宠助佑你；答：阿们。

赦免你的罪，拯救你，并减轻你的病苦。答：阿们。

 

5. 祷词  

( 然后神父请参礼者一齐祈祷：)

请大家祈祷：我们的救主基督！求你以圣神的恩宠，医好病人的虚弱，治愈他的创伤，赦免他的罪过，消除他身心的痛苦，恢复他身心的健康，使他克尽原有的职务。你是天主永生永王。

众：阿们。

成圣体圣血经 ( 跪下 )

他在被出卖的那天晚上，举起面饼，感谢、赞颂你，把面饼分开、交给他的门徒说：

「你们大家拿去吃 : 这就是我的身体，将为你们而牺牲。」

晚餐后，他同样拿起杯来，感谢、赞颂你，交给他的门徒说：

「你们大家拿去喝，这一杯就是我的血，新而永久的盟约之血，将为你们和众人倾流，以赦免罪恶。你们要这样做、来纪念我。」

 

祝圣后的欢呼 ( 起立 )

主祭：信德的奥迹

信友：基督，我们传报你的圣死，我们歌颂你的复活，我们期待你光荣地来临。

 

主祭：上主，因此我们纪念你圣子为拯救世人所受的苦难，和他神奇的复活、升天，并期待他再度来临，以感恩的心情，献上这具有生命的圣祭。

请垂顾你教会内的奉献，并接受这赎罪的牺牲；求你使我们藉你圣子的圣体圣血得到滋养，并充满他的圣神，在基督内成为一心一体。

愿圣神使我们成为你永恒的祭品，得与你拣选的人，首先与天主之母童贞荣福玛利亚、你的圣宗徒及光荣的殉道者 ( 和圣… )以及全体圣人，共同继承天国的产业；我们深信他们的转求，使我们不断地获得助佑。

上主，但愿这与你修好之祭，有助于整个世界的和平与得救，并使旅途中的教会 : 我们的教宗、我们的主教、所有主教、圣职人员，以及你所救赎的全体子民，在信德和爱德中坚定不移。

你既愿这大家庭侍立在你台前，求你也俯允全家的愿望。仁慈的圣父，求你使散居各处的子女，和你团聚。

求你恩准我们的祖先、已亡的亲友，和所有在你宠爱中去世的人，进入天国。我们也希望将来在天国永远同他们共享你的光荣。以上所求是靠我们的主基督，你藉着他赐给世界一切美善。

 

全能的天主圣父，一切祟敬和荣耀，藉着基督、偕同基督、在基督内，并联合圣神，都归于您，直到永远。

信友：阿们。

 

四、领圣体礼 

主祭：我们既遵从救主的训示，又承受他的教导，才敢说：

全体：我们的天父，愿你的名受显扬；愿你的国来临；愿你的旨意奉行在人间，如同在天上。求你今天赏给我们日用的食粮；求你宽恕我们的罪过，如同我们宽恕别人一样；不要让我们陷于诱惑，但救我们免于凶恶。
