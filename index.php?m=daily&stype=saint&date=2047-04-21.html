复活节庆期第八日(慈悲主日)

　2000年圣年的5月5日,真福教宗圣若望保禄二世发表了《慈悲宽仁》通谕(法令), 在罗马礼仪的弥撒和日课中定下: 复活节后首主日为【救主慈悲主日】。感谢真福教宗, 把救主耶稣基督的慈悲带给了普世教会。

　　　在圣女傅天娜的《日记》里, 耶稣至少十四次, 要教会正式建立一个《慈悲节日》: “我要有一个慈悲庆节。我要妳用彩笔, 绘画这张慈悲圣像, 在复活节后的第一个主日, 隆重地祝圣; 要定该主日为慈悲庆节。”( 日记: 49, 280 ) 耶稣的圣意, 要向我们展示祂救赎人类的复活奥迹, 是与天主救主慈悲的奥迹息息相关。 

     这【救主慈悲主日】不但是一个特别敬礼天主无限慈悲的神圣节日,  而且是天主对世人慷慨普施恩泽的宝贵时刻;  因为在这节日去告解和领圣体的灵魂,  等于得了“第二次洗礼”的所有救赎恩宠,  那就是:“所有罪过及罪罚的完全赦免!  在那天,   我至溫柔的慈悲之最深處敞開了,   并向所有投奔我慈悲泉源的靈魂,   傾注汪洋大海般的恩寵.   在那天,   所有恩宠的水闸都开放了.   让所有灵魂都不必害怕亲近我,   即使他们的罪过红得发紫! ”( 日记: 699)。   获得【救主慈悲主日】的全大赦所需条件为:  完全弃绝罪恶,  在当天或前后的三星期内办告解,  并领圣体圣事,  为教宗意向祈祷,  以及参与【救主慈悲】敬礼和祈祷, 或行其他方式的善工  ( 详情请参阅以下的附录 )。

    主耶稣藉圣女的文笔, 苦口婆心地, 一再劝诫我们:“尽管我为他们忍受了悲慘的苦難,灵魂依旧丧亡。 我现在给他们救赎的最后希望, 那就是: 我的慈悲庆节。如果他们再不崇敬我的慈悲,  他们将永远丧亡! 我慈悲的秘书, 要写下来, 并告诉所有灵魂我的大慈大悲, 因为恐怖的那一天, 我公义的日子近了! ”( 日记： 965 )

     耶稣嘱咐圣女, 要在慈悲节日前,  从受难节星期五开始,  一连九天,  虔诚诵念《慈悲串经》。  祂说:“透过这九日敬礼,  我会赐给灵魂一切所需要的圣宠。”( 日记:796 ) “ 我切愿妳,  在这九日内,  把众灵魂引进我慈悲的泉源中;  好使他们从那里吸取力量,  振作精神,  更得到面对生活中各种艰苦的恩宠,  尤其是临终时所需要的任何圣宠!  每一次,  妳要把一种不同的灵魂,  带领到我的圣心中,  将他们浸润在我慈悲的汪洋大海中。 我将带这些灵魂到我天父的屋子里去!  妳要在今世及来生都这样做!  我决不会拒绝他们所需要的任何恩宠!”( 日记:1209-1229 )  在这 “慈悲九日敬礼”中,  耶稣要我们, 每一天分别引进九类不同的灵魂: 全人类特别是罪人,  神父及修会人士, 热心及忠贞的人, 外教人及未认识基督的人, 散播异端及分裂教会的人, 良善心谦的人及小孩子, 恭敬光荣天主慈悲的人, 炼狱灵魂, 和冷淡灵魂 ( 日记:1209 - 1229 )。

     救主慈悲敬礼的最终目的, 是当我们领受了救主耶稣的慈悲后,  自己也要成为像祂一样慈悲的人。 要尊重关爱邻人如同耶稣,应当怀着如同亲近耶稣一样的爱情,并以恭敬圣体般的恭敬态度服务他们（日记: 285）。                    

“ 我要妳因为爱我的缘故, 实践慈悲的行为。 妳应当时时处处慈悲地对待邻人。不要畏缩不前,或找借口来推却责任。我现在给妳三个慈悲地对待邻人的方法: 第一是具体行动; 第二是透过语言;  第三是借着祈祷。这三种方法,都在不同的程度上,完全概括了慈悲的全部意义。因了这些慈悲的行为,才能毫无疑问地证明妳对我的爱。 透过了这三种行为, 一个灵魂才能光荣和恭敬我的慈悲。 当然,复活节后的第一个星期日就是慈悲庆节,但还应该有慈悲的行为。 我要求人们透过庄严地庆祝慈悲庆节,及崇敬慈悲圣像,来恭敬我的慈悲。 借着这圣像,我会恩赐灵魂无数的圣宠,并时时刻刻提醒世人,我慈悲的要求,因为如果沒有善行, 即使有最坚強的信德也无济于事。”(日记: 742 )

耶稣告诫我们: “ 没有任何灵魂能成义成圣, 除非他以信心投靠我的慈悲, 这便是为什么复活节后的第一个主日, 应当定为慈悲庆节! ”(日记:570)。 愿我们同心同德,为自己,为家庭, 为我们的国家,  为我们的教会, 虔诚祈求慈悲的救主耶稣,对我们大发慈悲。 希望大家每天恳切地诵念《慈悲串经》,默想主的苦难,勤参与弥撒和领圣体,  恭敬圣体和在圣体中的耶稣圣心;在困难的时候,不要忘记说: “耶稣,  我信赖祢! ”祈求痛苦之母,圣母玛利亚,她与耶稣同甘共苦。让我们一起庆祝欢呼: “ 主复活了!  阿肋路亚! ”

 




圣安色莫(坎特伯里总主教圣师 St. Anseln)

圣安色莫于1033年诞生，原籍意大利奧斯答城，家道富有。15岁时就有志弃俗修道，父母不准。未几，安色莫结交了坏人，贪恋世俗，那时慈母去世，父亲性情不好将他逐出家门，不愿与他见面。

后来安色莫深悔已罪，发奋读书。诺曼田白格隐修院有一位名师开班授课，安色莫去该院读书。三年后(1060年)正式入院修道。

1063年，良方升任关城圣斯德望修院院长，所以白格修院副院长一职由安色莫担任。有些修士见安色莫年纪太轻，一开始不肯服从他的管束，可是圣人用诚恳的态度，慈爱的精神，感化他们，到后来，全院修士都心悦诚服地服从他的领导。

安色莫博学多才，对神哲学尤有深刻的研究。成了当时最有名的神学家，也是“士林哲学”的创始人。他发表了几本有价值的著作，用形而上学的理论证明天主的实有，研究天主的性体，人的自由意志，罪恶的来源。

安色莫对青年教育问题所持的见解与现代教育理论，相当符合。他主张给予青年自由发展特长的机会，不应加以过分的束缚。

安色莫担任白格隐修院副院长为时15年。1078年升任该院正院长。安色莫经常前往英国，一来因为修院在英国置有房产，二来因为前任院长良方在英国坎特伯里任总主教。良方逝世，国王路福霸占主教府财产，禁止教会选任新主教。过了一个时期，路福患了一场重病，害怕天主降罚，不敢坚持原来主张。安色莫众望所归，当选坎特伯里总主教。

安色莫以年老体弱为理由一再推辞，神职人员强拥他到大堂。安色莫见民意难违，勉强接受圣职。

路福王恢复了健康，故态复萌，向主教索取巨款，安色莫严词拒绝。路福想游说教宗免安色莫，可是教宗早已洞悉路福的奸计，派钦使颁赐“羊毛带”给安色莫，以示嘉奖。

安色莫拟赴罗马面谒教宗。路福不准他出国，后恐吓安色莫说：“你离任后，主教府的财产，将全部充公。”安色莫不为所动，启程上道。中途与克吕尼修院院长圣许格、里昂总主教许格会晤。他到了罗马，将国王的不法行为奏明。教宗对他加以慰勉，写信劝路福尊重主教的职位和权利。安色莫在居罗马期间，为了创作一本关于圣子降生奥迹的不朽名著。

由于路福横加阻挠，安色莫无法返回，在罗马暂住。他一度申请辞去总主教职位，度隐修士的生活，但教廷不批准他。

那时，希腊教会对“圣父圣子共发圣神”的信德道理，见解错误。教宗召集会议，安色莫向大会阐述圣教正统道理，获得全场人士的一致拥护。

路福去世后安色莫返回英国。某地主教出缺，新国王亨利认为他有权推选继任人选，安色莫坚决反对，坚持主教须按教律选任。双方的冲突，一天比一天尖锐。安色莫亲往罗马朝观教宗，教宗支持安色莫的立场。亨利大怒，禁止总主教返任，并宣布主教府收益一律充公。安色莫决定给国王绝罚处分，消息传到英国，全国震动。亨利见势不佳，立即表示让步，放弃推选主教的非法权利。从这时起，国王与总主教的关系才逐渐融洽，亨利出国期间，甚至特委安色莫暂持国政。

安色莫治理教务多年，积劳成疾。1109年，在坎特伯里逝世。

安色莫性格温和，待人诚恳，对穷人尤为爱护。朝野人士，不分贫富贵贱拥护他。他是禁止贩奴运动的先锋，1102年安色莫在西奇召开全国教圣教会于1720年敕封安色莫为圣师。
