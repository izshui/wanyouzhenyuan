什么样的私启示可信？从《朝拜耶稣圣心一小时》的问题谈起
纳爵之盾   2020-10-22   9939

在特别敬礼耶稣圣心的六月，有一份来源不明的类似“私启示”的祷文《朝拜耶稣圣心一小时》在微信上疯传，其中大意是“为赔补圣心被伤害的痛苦，为灵魂求得悔改的恩宠”。看似充满信仰热心的祷文，其中不少内容需要警惕。除此之外，“纳爵之盾”也列出几个以私启示之名公然诋毁攻击教宗，蔑视教会训导的公众号（点击参阅），这些所谓的“私启示”极具迷惑性，更需要各位神长教友提高辨别力。

 

那么，什么是私启示？它与公启示有何区别？私启示有何价值和功用？如何分辨私启示的可信度？本文将基于对上述问题的回答，来审视《朝拜耶稣圣心一小时》祷文。

 

 // 一、什么是公启示和私启示？ 

 

基督宗教是启示的宗教。“启示”（revelation）的本义是“掀开帐幕”，即对事物真相的揭示。用在天主身上，指的是天主的自我通传。天主自我通传的动机，是“因祂的慈善和智慧，乐意把自己启示给人”，目的是“使人认识祂旨意的奥秘”。启示的具体内容是“关于天主以及人类得救的深奥真理”[1]。至于启示实现的过程和中介，《致希伯来人书》这样总结：“天主曾多次并以多种方式，借着先知说过话以后，‘在这末期借着圣子对我们说了话’。”（希1：1）

 

在旧约时代，天主拣选了以色列民族，派遣先知向他们启示天主的话，并为圣言降生做准备。在“末期”，即圆满时期，耶稣基督来完成了一切启示，圆满的启示就是启示者耶稣基督本身，因为祂是“道路、真理和生命”（若14：6），是天主圣言。耶稣启示的对象也不再是某个特定的民族，而是指向全人类。“在他以后，再没有其它启示了。”[2]由旧约所开启，耶稣基督所完成的启示，便是所谓的“公启示”。

 

从耶稣基督完成救赎工程升天之后，直到祂再来的整个期间，公启示之信仰真理，托付于宗徒的继承人即众主教来管理、保存和传递。不过，尽管启示已告完成，却尚未完全阐明，还有待基督徒在教会训导的指引下，更深入地领悟其内涵并进一步得以显明。

 

除公启示之外，在历史中也出现了一些所谓“私启示”。“私启示是私人所得的启示，内容不超出公共启示，他人也没有责任接受。私人的启示往往产生在特殊时代，间接有助于信仰生活。如：耶稣圣心显现给圣母往见会修女玛加利大·亚拉高（Marguerite-Marie Alacoque, 1647-1690）即是一例。”[3]

 

 

 // 二、私启示的价值和功用 

 

如果说公启示在旧约时代是借一个民族且针对一个民族而进行，在新约时代是借耶稣基督，针对普世而完成，那么，私启示，顾名思义，是借个别人，针对某事、某人、某地或普世而传递的信息。既然启示已经圆满，就无所谓增加或改善。因此，任何私启示在本质上都是“多余的”，既不属于信仰的宝库[4]，也没有当信的义务。

 

当然，“多余”并不并意味着无用。假如其讯息不与公启示和教会训导相冲突，私启示有什么价值或功用呢？它唯一的价值，是把人引向公启示，因而其功能是辅助性的，即“协助人更圆满地履行基督的启示。”[5] 在《上主的话》劝谕中，教宗本笃十六世明确指出：

 

“私人启示的价值，在本质上与那唯一的公共启示不同，后者要求人发出信德……私人启示是对信德的一个辅助，它是否可信，全在于它能否引人归向那唯一的公共启示中。教会认可一个私人启示，基本上就是说其内容没有任何相反信仰或伦理的成份；可合法公开这启示，信友也可以审慎地去接纳。私人启示可让人注意到新的重点，兴起新的或加深已有的敬礼。它也能有某些先知特性（参阅得前5：19-21），可成为有用的辅助，让人在某些特别时刻更明了和实践福音。因此，不能掉以轻心。这是向人提供的辅助，无须硬性规定采用。无论如何，私启示必须滋养信望爱三德，因为信望爱才是为众人得救的永恒途径。”[6]

 

// 三、如何分辨私启示是否可信？

 

教会之所以对私启示持有十分审慎的态度，并提醒教友不能掉以轻心，是因为它有可能是虚假和邪恶的。从来源上讲，自我认定的“私启示”，或来自天主，或出于魔鬼，或是人为的。只有出自天主的私启示，才是严格意义上的“启示”，才是真的和好的。出自天主的私启示，有时借天使或圣人，特别是圣母玛利亚来显现和传递讯息，如教会承认的露德圣母与法蒂玛圣母。

 

有些所谓的“私启示”，其实是出于人自己的认知、想象或幻听幻觉等心理问题，乃至谎言、不良意图，它们一定不是真的，但不一定是坏的（如出于心理问题的幻觉）；若是出于不良意图而胡编乱造，引人偏离基督和教会，则是坏的。出自魔鬼的伪装的“私启示”，一定是虚伪和邪恶的。

 

私启示从方法上讲，有外在与内在之分。外在私启示居多，一般是以直接“显现”的方式呈现。比如：耶稣对玛加利大与傅天娜两位修女的显现，以及露德与法蒂玛圣母的显现，都是外在和真实的，不是幻象；显现者传递的信息也不是通过内在感受，而是以真身肉嗓、实际感官被领受的。内在启示，也称内心交谈，是指一个人在进入某种灵修的状态后，天主在其内心对他/她发出的声音与召唤。

 

 

既然只有出自天主的私启示才是真的和好的，因此，诸如那些出于人自己或魔鬼的“私启示”，都应当予以抛弃。那么，如何辨别一个私启示是否出自天主呢？如何确定一种内心交谈是天主的话，而不是人的心理作用或魔鬼在作祟呢？

 

既然私启示是为引人归向公启示，而圆满的启示不只是一些“内容”，而是生活的一位——耶稣基督，那么正如教宗本笃十六世所指出的：“判断私启示真伪的标准，就是看它是否引导人走向基督。若它引导人离开基督，就必定不是出自圣神，因为圣神引领我们更深彻地进入福音，而不是远离祂。”[7]这个唯一的分辨原则，是从果子来判断树之好坏的标准。正是依据这一原则，信理部在1978年颁布《有关分辨所谓启示之显现的进行方式的标准》[8]，文件具体列出积极与消极两个层面的分辨标准。

 

···········································································································································

 

在积极层面上，判断一个所谓的私启示是真是伪，要看以下几点：

（1）其讯息是否符合伦理？

（2）启示接收者的心理、伦理是否正常？是否服从教会权威？能不能回到正常的信仰生活中？

（3）在神学和灵修上，启示讯息是否无误？

（4）该私启示是否带来健康的敬礼，结出丰硕和持久的灵性果实（如祈祷、悔改、爱德的见证等等）？

 

在消极层面上，可从以下几点来判断：

（1）有明显的事实上的错误，如异想天开的想象；

（2）把错误的教义投射在天主、玛利亚或圣人身上；

（3）有明显的不良意图；

（4）“启示”接受者或其追随者有严重的不道德行为；

（5）“启示”接受者有心理或精神问题。

 

········································································································································

 

上述分辨标准非常清晰且全面，足以帮助我们辨别一个所谓的“私启示”是否来自天主。据此，我们至少可判定，诸如“小德兰”（之前的马甲为“小德兰书屋”）、“耶稣说你们悔改吧”、“法蒂玛之秘密2”和“爱的呼声”等公众号的问题：

 

1、在教义上，它们所依据或解读的所谓“私启示”讯息，明确反教宗和教会训导；

2、在态度上，它们的文字充满恶意、攻击、尖酸、诽谤和谎言，完全相反圣神的果实（参阅迦5）；

3、在事实上，它们歪曲原私启示的信息以作攻击教宗之用，意图明显不良；

4、在伦理上，它们制造分裂和敌意；

5、在影响上，它们蓄意让人脱离建基在伯多禄磐石上的教会，从而企图让人远离基督。

 

总之，从各方面讲，它们所依据或歪曲的“私启示”讯息，都不是出自天主。因此，再次奉劝教友不要关注和转发它们的文章！同时，再次呼吁主教和堂区司铎们提醒教友小心这些不良公众号。

 

 // 四、《朝拜圣心一小时》祷文存在的问题 

 

与上面几个毒害明显的公众号不同，祷文《朝拜圣心一小时》的问题，在于有可能让祈祷者在无意识中产生不正确的天主形象和看待他人的态度，并且这种危险不易觉察。我们举出上面几个不良公众号和这篇祷文，除了是为实际地指出它们的问题外，也是为提供不同范例，帮助信友学习分辨私启示。

 

 

《朝拜圣心一小时》祷文分甲乙两部分，甲为“我们向耶稣圣心的祈祷”，乙为“耶稣圣心给我们灵魂的对话”。对话的内容有点模仿《圣母给心爱神子的话》，以“耶稣圣心对我的灵魂说”开始，属于典型的内心交谈。分辨一种内心交谈是否是真的私启示，同样需要依据上面提到的分辨标准。针对这份祷文，我们只需从原则上看它能否把人引向基督所启示的福音和天主的形象即可。

 

我们先展示一下该祷文的内容。在甲部分里，祷文邀请人默想倍受凌辱、无家可归的耶稣正“心诚意切，特地来投奔你，向你要求几件事情。什么事情呢？就是世人犯罪得罪祂、不爱祂，你应当加倍地爱祂，为赔补世人的负心之罪”。接着，祷文说：“本来照着天主的公义，很可以惩罚这些人们，如同从前洪水灭世、火烧淫城等等，以灾害来处罚这些亵渎圣事的恶人。但吾主圣心始终慈悲为怀，可怜世人，不忍这样对付世人。虽然他们不悔改，只要你肯代行补赎，吾主便宽恕了他们。”

 

乙部分的第一段，以耶稣的口吻说：“你代替那些凌辱我的人作补赎。 他们忘恩负义，罪大恶极，他们明知故犯大罪，明知道我是爱他们的，可是他们偏偏离开我。”

 

上面两段话，概括了整篇祷文的大意：首先，有人罪孽深重、忘恩负义，令耶稣伤心，该受永罚；其次，耶稣慈悲为怀，怜悯他们；最后，基督徒要为他们做补赎，上慰圣心，下求悔改。接下来乙的部分的对话，都是围绕同一思想展开。在整体上，与其说这是一篇祷文，不如说是一篇针对各种类型的“罪人”的檄文，文字中营造了浓厚的对抗氛围。典型的段落如：

 

“吁！耶稣圣心，从前你在黑落德恶王面前所受的羞辱，比受死还厉害。可是现在一些自认为聪明的人，著书立说，污辱你的圣名，毁谤你的神圣，比起黑落德更凶恶，他们已濒临永罚的边缘还不知道。因此我恳求你，因你无限的仁慈和圣母的爱情，变化他们的铁石心肠，使他们及早回头归正吧！”

 

····························································································································

 

这篇祷文有两方面的问题，一个是天主的形象，另一个是看待他人的态度。祷文虽然不断提及天主的慈悲和对罪人的怜悯，实则在侧面强调了天主的公义和人应受的惩罚，把天主的慈悲依附于对正义之罚的撤销，把天主的形象限定为“赏善罚恶”的主宰。然而，“天主是爱”（若一4：8），祂的慈悲在本质上是无偿的，祂对罪人的宽恕乃基于无所求的大爱；基督来，不是为基于正义进行审判，而是为在爱内施行拯救（参阅若12：47）。

 

教宗圣若望保禄二世在《富于仁慈的天主》通谕中指出：“单讲公义是不够的，天主愿以更有力量的爱全方位地塑造人。若单单强调公义，会导致公义本身被否定；慈悲是公义的实现形式。”[9]耶稣在显现给傅天娜修女时说：“但愿没有灵魂害怕到他跟前来，即使他们的罪红得发紫。”[10]天主以慈悲对待罪人，是为让他们回到天父的怀抱。而我们的使命，是成为慈悲的工具，向每个人显示天主的仁慈，把光明引入人心，在每一个人心中播撒天国的种子，让他们学习远离罪恶，接纳天主；我们要做引导者，而非审判者。

 

 

《朝拜圣心一小时》的第二个问题，是祷文的诵读者不是以罪人的身份祈祷，而是显得有点事不关己地“怜悯”或更好说批判伤害耶稣圣心的罪人——他人。当我们在阅读或颂念这样针对“别人”的祷文时，很难反省自己，忏悔己罪，因为焦点都集中在对“别人”的控诉上。若长此以往，我们的心态就会改变，放眼望去，遍地都是堕落、自私、忘恩负义的“罪人”，而自己却像一朵出淤泥而不染的莲花，只能靠“我”的补赎来安慰“他们”所伤害的耶稣圣心。如此，我们很难透过这样的祷文用天主慈悲的眼光看待别人。它所呈现出的，其实是“荡子回头”比喻中的大儿子的心态，在某种程度上不利于个人灵修的健康发展。

 

 // 结 语 

 

由此可见，哪怕是像《朝拜圣心一小时》祷文这种无实质性问题的私启示，也有可能把人带偏。而诸如我们列出的那几个公众号所依据的所谓“私启示”，则更是虚假和恶意的。如果连那些被教会承认的私启示，我们都没有当信的义务，其它不被教会承认或被歪曲的私启示，就更不值得关注和相信。总而言之，在教会信仰里，私启示是有之不多，无之不少的东西。所以，为保持纯正信仰，不陷于危险，正如教会训导不断提醒的，面对任何私启示，我们都当十分谨慎。

 

渴盼在灵修上精进者，应该从教会悠久敦厚的传统中汲取营养，而不应在求新轻信中迷失。敬礼圣心可有多种方式，如默祷、朝拜圣体；在经文方面，有教会所认可并推广的慈悲串经。这些都是我们可用以敬礼圣心的方式。作为教友，我们在灵修上不必刻意追求标新立异，尤其要对网络上流传的各类讯息持审慎的态度。
