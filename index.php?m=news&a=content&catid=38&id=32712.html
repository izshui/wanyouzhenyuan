中国大陆教会的神恩复兴乱象
纳爵之盾   2020-10-29   1.88万

 

神恩复兴运动，指基督宗教信仰生活中，有关神恩的更新运动，又称为“圣神同祷会”。本文所说的神恩特指由神恩复兴运动中得到的内心感动、热诚、说异语、图像等，而不是指由领受圣事而得的宠爱、宠佑和特定的圣事恩宠。

 

天主教神恩复兴运动是受基督新教“新五旬运动”的影响，于1967年在美国开始，后逐渐发展壮大， 1975年教宗保禄六世为此成立了“天主教神恩复兴国际办事处”，1995年更名为“天主教神恩复兴国际服务处”，并对此运动给予嘉奖和鼓励。随后的教宗们也给予了不同的肯定。

 

中国大陆教会的神恩复兴运动是在上世纪90年代初期由台湾传入，先是在北方的个别教区中的个别堂区开展，2000年后以更快的速度发展到各地不同的教区和堂区。但因其中出现了许多乱象，因此被许多教区的教长或本堂所禁止。

 

本篇小文是笔者根据自身领受神恩的经验和所经历的有关神恩的种种，尝试从教友和神职人员两个角度分析一下其中的乱象，以防微杜渐，免覆辙重蹈。

 

教 友 篇

 

 

据笔者所知，神恩复兴运动在上世纪九十年代中开始在北方的正定教区范围内开始慢慢活跃，后传播到四周的其它教区。在大陆神恩复兴运动的早期阶段，核心团体还受到过天主教华语界神恩复兴大师王敬弘神父的督导。

 

在实际操作过程中，因为大部分参与的教友缺乏对神恩本身和复兴运动的全面理解，而产生了诸多使人啼笑皆非的神恩“乱象”。

 

在一些地方，该活动不但没有起到积极的效果，甚至让人更加抗拒和保守式地故步自封。如果放在属灵争战的大背景下去理解，这运动在某种程度上非但没有光荣天主拯救人灵，反而成了魔鬼利用人性软弱而迷惑教友的工具。下面提出一些“乱象”，大家可以借鉴以反省自身。

 

// 1. 由神恩所发的骄傲与分裂

 

有些教友的确在神恩祈祷中得到了不同的神恩（舌音、异语、先知话、解释异语、图像、身心灵治愈等），但这些神恩在不经意中让人慢慢陷入骄傲的境地，觉得自己比他人更圣善、更得天主欢喜。如果再有人因他们的神恩来找他们排忧解难，就更会让他们觉得自己与众不同，甚至不听从神长的引导和指令，我行我素，以致分裂教友团体。

 

天主给予某人神恩是为建树教会和团体，不是因为此人圣德非凡。人应因为领受神恩经验到天主，从而不断更新自己的灵性生命状态，防止骄傲的诱惑而成己达人。关于这一点，大家可从《格林多前书》第十二章看到圣保禄宗徒对神恩的详细描述。

 

// 2. 由不纯动机所结的恶果

 

由于在神恩祈祷中得到的恩赐（得到指引、治愈、启示、舌音异语等可见神恩），教友们在做此种祈祷时，难免有彼此攀比或因没有得到神恩而失落的心态。

 

如果在准备神恩祈祷时，不是为了服务教会或团体，不是为了更渴慕天主的触动，不是为更好承行天主的旨意，大多数人就会下意识地随着自己的本性欲望而追求外在的记号。

 

有的人本没有得神恩，因不想被人“笑话”，便假装得了某种神恩，实乃不该！要知道，神恩是天主圣神自由赐予信友服务教会的不同力量，以开放的心随天主的圣意就好。

 

// 3. 由不善用神恩而结的恶果

 

由于教友们缺乏对自己灵性生命的深度认识，即使得了某种神恩，如果不能在灵性生命上继续划向深处，反而满足于此神恩，就会慢慢陷入恶神的圈套。比如借神恩的影响卖东西挣钱，要想病好，必须买我的东西。

 

越是经验到神恩与黑暗势力交战的人，越是要分外小心，更要不断分辨天主善神的推动，否则会很容易走偏。因为魔鬼很清楚，让他们跌倒会起到事半功倍的效果。

 

// 4. 拥有神恩不是一劳永逸的

 

很多人把“说异语”当成了在圣神内受洗的标记，其实是不恰当的。

 

如果一个人的教友生活没有把神恩、读经、祈祷和爱德服务四者完整结合起来，其实就不算真正地在圣神内受洗或更新，也就很难结出相应的在圣神内的果实。当然，整合四者会是一个漫长的过程，因领受某种神恩而开始生命改变的步伐就是一个美好的开始。

 

// 5. 把神恩当成了“万灵丹”

 

有的教友领受了图像的神恩，从此不断有人来问东问西，与算命先生的营生类似。

 

这种把神恩窄化，甚至让人误解信仰真谛的行为，需要万分警惕。因中国的民间迷信文化也在深深地影响着教友们的信仰和认知，切勿使神恩沦为类似“求签问卜”的境地。

 

// 6. 热诚有余，明智不足

 

有些教友出于好心，愿意让更多的弟兄姊妹或领受神恩，或得到治愈（心灵和身体皆可有），或经验神奇，便不加分辨地、来者不拒地一起做神恩祈祷。

 

殊不知，如果对方没有足够的心理、身体和灵性的准备，不但起不到好的效果，反而会让对方的状况更差。有的本来没有什么不正常，因为做了类似的祈祷，反而精神崩溃、神魂颠倒，结果适得其反。

 

很多堂口在神恩复兴运动过后，虽激情慢慢趋于平淡，但若灵性生命质量有了提升，教友生活有了改变，这都属于正常情况。如果激情不再，灵性生命也没有积极改变，则属不正常。就更不用说那些因为神恩复兴运动而为堂区带来的纷争与不睦了，那完全是人们误解或滥用了天主所赐的神恩的缘故。

 

在教会历史上自古就有神恩滥用的“乱象”，但天主仍然在历史的各个阶段，透过不同方式的神恩给予，感动如云般的见证人——圣人圣女们——以各自不同的神恩继续祂的救恩工程。

 

大家可以学习圣人们善用神恩的榜样，以更好发展自己的神恩。在做神恩祈祷时，最好找一位明智、热诚、向天主开放的修道人定期做神恩的分辨和灵修的辅导，以免在不知不觉中陷入魔鬼的圈套。毕竟魔鬼特别“喜爱”那些拥有某种神恩的人。

 

各神恩小团体需要跟本堂神父密切沟通，并受其监督，最好在神父的带领下学习有关神恩的教会文献和相关书籍，以正确认识教会有关神恩的教导和描述。

 

神 职 篇

 

 

面对神恩复兴运动中的乱象，神长的因素及态度无疑是举足轻重的。正当的态度和处理方式定会让乱象尽快消失。但不幸的是，一些神长的不当方式，反而让乱象更加地“乱”，甚至成为乱象的一部分。

 

// 1. 神长的负面印象

 

反对神恩复兴运动的神长大都没有从神恩复兴运动中得到过感动，甚至没有参加过。他们看到的是大家祈祷时的肢体舞动、大喊大叫、哭笑无常等，这些与教会传统的圣事礼仪和祈祷方式大相径庭，因此极为反感，不能以开放的心接受圣神的感动，也因此与神恩无缘。

 

有些神长也因为缺乏实际的神恩体验，而一味地否定此运动，实为可惜。

 

在梵二《教会宪章》第4号、《司铎职务与生活法令》第9号、《教友传教法令》第3号和《教会传教工作法令》第4号中都可找到有关神恩的教导，并向大家说明了应以何种态度接受及如何运用神恩，神长们可以做相关了解。

 

// 2. 神长的武断决定

 

面对神恩复兴这个“新兴”事物，神长们应持以开放的心态，承认圣神的确透过此运动在教会内工作，因其不但得到教会当局的认可，也有其可见的积极福传效果。神长们固然有保护、引导信友灵性生命的责任在身，但不能因噎废食，或用社会上流行的手段，一刀切地发布“凡我堂区的教友不得参加，不得在圣堂内举行神恩祈祷”等禁令。这不但不能引导信友正确地认识、对待此运动或各种神恩，更使他们因疏于照顾而陷入于恶神的诱惑和攻击中。

 

// 3. 神长的人性软弱

 

这一点相当确实，又相当地被忽略，以至于人们可能会忽略此一现象：没有领受某些神恩的神职也会嫉妒一些确有神恩的信友。

 

一些情况下，恐于个人灵魂状态的问题，个别神职甚至害怕面对那些具有先知之恩的信友，也有神长难于接受被信友覆手祈祷，好似自己矮了一截。毋庸讳言，我们的妒意、骄傲和矫饰常常阻碍圣神的工作，魔鬼也会利用这些来操控人心的自由。

 

//  4. 神长的神学匮乏

 

凡以开放的心参加过神恩祈祷的神长，很多得到了某些神恩，虽热诚于神恩祈祷，但却缺乏必要的神学造诣，因此也有了一些怪异、不符合教会正统神学思想的神恩教导。

 

例如，一位教友车祸，一位有神恩的神父去探望并做神恩治愈祈祷，说这车祸是因其祖父曾犯罪的连带结果。这位神父此般描述，既缺乏神学支持，又没有实践基督爱德的精神。以此给教友带来的属灵伤害是不可估量的。

 

// 5. 神长的“混合礼仪”

 

一些热衷于神恩祈祷的神长不遵守弥撒或圣事礼仪规则，在弥撒里随意添加神恩祈祷的元素，让那些热衷于神恩祈祷的教友分不清礼仪的轻重，让那些没有参加过的教友们无所适从。

 

例如，有些神长在教区或堂区推行治愈弥撒、神恩弥撒等，弄得礼仪不伦不类。另外，神恩祈祷因受基督新教影响，在进行中有诸多的呼喊，如“阿肋路亚”、“阿们”、“耶稣基督”等，与传统的教会祈祷方式不同，因此给很多教友带来不适，甚至厌烦。神长可以从教廷《治愈祈祷训令》中找到指引，也需要告知且引导不参加的教友们“这些呼喊”并不是基督新教的专利。

 

// 6. 神长的过度推动

 

凡受惠于神恩祈祷的神长，大都切愿自己辖区的教友都能参加此类的祈祷。这种出于好心的牧灵举措，使那些不适合做神恩祈祷的教友（指那些体弱、有潜在心理创伤和疾病，有重大灵魂缺口的人）陷于精神崩溃或被恶神干扰的危险；使那些内心不明白、情感上抵触神恩祈祷的教友更加不能从堂区的活动中受益，从而使堂区的教友分为神恩派和“冷谈”派，前者得到神长器重，后者被冷落。

 

有些神长令教友们在晨祷时用神恩祈祷，其他大部分的教友却被疏忽或排挤，只能回家自己念经祈祷。更有甚者，对于教区主教关于限制或整顿不良神恩现象的劝告或公函，置若罔闻，拒不听从。这便正中恶者下怀。

 

为排除教会内的“神恩”乱象，使天主圣神透过神恩祈祷有利地推动教会的发展，建议各位神长不要怕惧此运动在运作过程中因不谨慎所带来的各种问题，要积极学习有关神恩的神学教导，甚至亲身参与进去，以正统的教会神学引导大家，好借助神恩的热火推动教会的各种事工。

 

纵观国内福传活跃的教区，其传道员大都有过类似的神恩祈祷经验，在灵性生命被改变后，勇敢地走出去福传作证。

 

为保障神恩祈祷健康进行，可在教区内成立“神恩复兴小组”，以特别服侍和指导神恩复兴运动。例如香港教区的“圣神内更新团体”就值得借鉴。

 

对于不良的神恩运用现象，教区教长可根据当地的情形，制定适当的规则。一个有信德、明智、开放的神长，不会因怕惧神恩祈祷中的个别不良结果而不允许教友们参与此类祈祷，他会重视发展教友们的不同圣召，并以自己的神学造诣来帮教友分辨不同的神恩和神类，使其各尽所能来建树主的教会，使教友们的灵性生活更加多彩缤纷。

 

总之，既不要“迷信”神恩，也无需视神恩为洪水猛兽。保禄宗徒曾极力呼吁：“你们要追求爱，但也要渴慕神恩，尤其是要渴慕做先知之恩”，（格前 14：1）只要我们正确认识并运用神恩，它在为我们的灵性生命带来益处的同时还能建树教会。愿主名受显扬！
