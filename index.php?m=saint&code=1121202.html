圣妇若翰纳•方济加•尚达尔
圣母往见会创办人 St. Jane Frances de Chantal -1641
 12月12日

圣妇若翰纳原籍法国第茸，父亲彼尼是第茸议会主席。若翰纳自幼丧母，由父亲抚养成人。彼尼非常热心，他的子女都受了良好的宗教教育。若翰纳领坚振时，取方济各为圣名。

若翰纳年二十岁，嫁尚达尔男爵为妻。男爵是福女恩培利的后裔(福女恩培利的瞻礼也是今天)，若翰纳婚后很是幸福，生了一子三女。夫妇二人，热心敬主，度标准的基督徒生活。

结婚后第九年，尚达尔男爵出外打猎，被朋友不慎射伤，数日后因伤重不治而终。若翰纳痛不欲生，意志消沉。老父来函瞩她拿出勇气，抚育孤雏。

若翰纳全力救济穷人，她用一部分时间教育子女，一部分时间祈祷默想，一部分时间从事穷人福利工作。她完全宽恕那个开枪不慎射死她丈夫的人，在一切事上帮他的忙。

若翰纳求天主赏赐她一位明智的神师，指导她承行天主的圣意。有一天，她正在这样祈祷，仿佛看见面前站着一个人，后来若翰纳在第茸初次见到圣方济各·撒肋爵的时候，才想起那天在神视中所见的人，和方济各·撒肋爵的面貌服饰一模一样。

有一次，若翰纳在树林中，仿佛看见前面有一座小堂，她想走进去祈祷，可是走了好久，那座小堂总是可望而不可即。她知道这是天主给她的教训，使她知道，我们应当全心爱天主，不应掺杂自私的心理。

若翰纳丧夫后第一年，父亲接她带众儿女回家暂住，守孝期满，她搬往蒙德隆夫家。尚达尔男爵的父亲是一个七十七岁的老翁，性情孤僻，待媳妇很苛刻，可是若翰纳逆来顺受，从不出半句怨言。

1604年，圣方济各·撒肋爵到第茸作封斋讲道，若翰纳也去参加。她一见了方济各，就认出便是当年在神视中所见的那个人，她知道方济各是天主赏赐给她的神师。

若翰纳请方济各作她的神师，方济各慨然允诺。从那时候起，若翰纳的神修善功有了适度的支配。若翰纳是一位家庭主妇，四个孩子的母亲，她不能抛弃基于个人身份应尽的一切事务。人们常说：“尚达尔男爵夫人常作祈祷，但是不使任何人感到麻烦。”若翰纳全心抚养子女，同时照顾病人，给重病人准备善终。

若翰纳原是一个性情刚强的女子，可是她和天主的圣宠合作，利用祈祷、克苦、忍耐熏陶自己的品性。方济各·撒肋爵禁止她做过分的苦行，也提醒她不要忘记自己还保有世俗妇女的身份，尤其是母亲的身份。他敦嘱若翰纳教养子女，应耐心开导，不可采用严厉高压的手段。

若翰纳早已有志出家修道，她对第茸的加尔默罗女修院的生活颇为向往。她征询方济各·撒肋爵的意见，方济各考虑了一个时期，向若翰纳表示，他有意创设一个新的女修会——圣母往见女修会。若翰纳大喜过望，可是这计划如欲付诸实施，还有许多困难需要解决。第一个问题是：她的老父是否肯表示同意？第二个问题是：她的四个子女怎样处置？第三个问题是：她的家产怎样处理？第一个和第三个问题都获得了满意的解决；关于子女的问题，若翰纳将长女出嫁，次女和三女暂时带在一起(次女不久患病去世，三女出嫁)，十五岁的儿子托父照管。

若翰纳将一切家事安排好了，挥泪离开家庭。她的儿子苦苦挽留她，躺在房门口，不让她出去。若翰纳硬着心肠，从儿子身上跨过去，到了大门，老父站在门口，若翰纳跪下，求老父祝福。父亲将双手覆在她头上，说道：“我不怪你，你弃家修道，我完全同意。我将你奉献给天主，天主叫你到哪里去，你就去。我很高兴，因为我知道你和天主在一起，望你为我祈求。”

方济各·撒肋爵在亚尼西布置院舍，1610年天主圣三瞻礼，修院正式成立，若翰纳和其他两位修女领受会衣。不久，又有十四位修女加入。那时候，修会还没有正式的名字。

方济各·撒肋爵给新修会拟订了会规，这篇会规大部分取自圣奥斯定会会规，会名是方济各早已决定的——圣母往见女修会。

方济各以谦逊和温和为修会的基本精神。他对若翰纳和修女们说：“你们应当无限制地修务谦逊的德行，你们应当经常发挥温和的精神。”方济各给若翰纳和往见会修女写了那本《爱天主》的不朽名著。

若翰纳领会衣后第二年，老父逝世，她回到家里，料理丧事。亲戚朋友劝她还俗：“为什么你将自己的青春埋没在修院里，快快把这件会衣撕成碎片吧。”若翰纳对于这些话，付之一笑。

方济各·撒肋爵在巴黎、里昂、马林、格肋诺、蒲治设立往见会分院。他请若翰纳主持巴黎分院的院务，她担任巴黎分院院长三年之久。

1622年，方济各·撒肋爵逝世，若翰纳痛失良师，悲伤莫名。1627年若翰纳的儿子奥斯定被派往作战阵亡，遗下寡妻和年仅一岁的孤女，这个女儿，就是未来的沙维尼夫人，十七世纪法国文坛最有名的女作家之一。若翰纳惊悉爱子阵亡，很勇敢地接受这个重大的打击，她将这个打击奉献给天主：“主呀!请你将违反你圣意的一切消灭，都烧掉。”

1628年，法国、撒伏约、毕爱蒙疫病盛行，往见会修女丧命者颇多，疫病传播到亚尼西，若翰纳不肯离城，她留驻修院，奋不顾身，侍候病人。

1632年起，若翰纳遭受神枯的考验，这是上主磨练他钟爱子女的一种方法。若翰纳勇敢地接受考验，苦尽甘来的日子，已经在望，她将获得天国永远的赏报。

1635年至1636年，若翰纳巡视各地分院(其时分院总数达六十五所)。1641年，法国王后邀她往巴黎，人们盛礼欢迎，若翰纳素爱谦逊，引为苦事。由巴黎回来，中途突患重病，1641年12月3日逝世，享寿六十九岁，遗体迎往亚尼西。

若翰纳于1767年荣列圣品。

圣味增爵对若翰纳的评语如下：“我认识的教友中，若翰纳是圣德最杰出的人物之一。”
