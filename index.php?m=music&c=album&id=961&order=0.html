                                                        #灵修生活
                                                    

                        

                            

                                
《属灵争战》是苦修神学的经典著作，是天主教灵修史上非常重要的作品之一，与隐修士多默·耿裨思的《师主篇》齐名。每一位基督徒都应该阅读，因为它指导我们修证完美圆满的成全圣德，及获得永生救恩的重要计策。这是16 世纪的灵修名著，是戴狄诺律修神职会会士，劳伦斯·司格布里神父(Dom Lorenzo Scupoli)的精心杰作。几个世纪以来，许多修行人依之修行成圣，著名的圣方济各·沙雷氏称之为:他的“黄金宝典”，是他“最喜爱的，最亲爱的书”。他在意大利帕多瓦大学读书时，亲自从司格布里神父手中得到此书。此后 18 年间，他常随身携带，每天依照书中的教导默想。书中的修行智慧，启迪他挚爱天主，如何过虔敬圣洁的灵修生活，使他最终成为伟大的圣人，及受人尊敬的教会圣师!他极力推荐此书，给所有在他指导下修行的人。盖恩夫人也在自传中，宣称她依此书灵修，在无尽的迫害与极端的痛苦折磨中，悟出天主的圣意和爱，她自愿终身虔诚地承行天主圣父的旨意，从而修证圆满的圣德！
