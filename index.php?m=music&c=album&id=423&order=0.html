                                                        #人物传记
                                                    

                        

                            

                                
圣类思公撒格（St. Aloysius）


青年主保圣类思公撒格生于一五六八年五月九日，出身意大利隆巴田贵族，父亲是加斯蒂利候爵佛郎德，在西班牙宫廷供职。


类思自幼养成每天念早晚课的习惯。到了七岁，除了晚课外，每天念圣母小日课，七篇忏悔圣咏和其他经文。念的时候，双膝跪地，下面不用垫子。


类思从孩提时代起，已知脱离世俗的虚荣，一心归向纯善至美的天主。根据他的神师圣罗伯多贝勒明及其他三位听告司铎的意见，类思终身未犯过大罪。


他对当时富贵人家的淫逸奢侈生活，非常痛恨，在圣母像前，自动发了终身守贞的愿。同时，为了约束自己，严守贞洁，每天学古代旷野隐士的榜样，用苦鞭抽打身体。他处处留意，非礼勿视，非礼勿言，态度端正，宛若天使。



那时候，类思读到一本描写耶稣会士在印度传教的书，他对这些离乡背井到远方传福音的战士非常钦佩，渴望步入他们的后尘。他先从讲解要理开始，利用暑假的机会，每天给附近的穷孩子讲要理。


类思勤操苦行，每星期守斋三天，仅吃面包清水，半夜起床，跪在冰冷的石板上，通宵祈祷。


两年初学期内，类思的言行完全符合一位修士的标准，修院的生活是有规律的，类思每天工作吃饭休息都按一定的时间，不再像以前一样随便，一天到晚，只顾祈祷默想，连睡眠吃饭，都忽视不顾。


一五九一年，罗马发生大瘟疫，丧命的不计其数。耶稣会办了一所临时医院，连总会长也亲自侍候病人。类思也在院里帮忙，他很愉快地照料病人，结果也染上了疫病。领了终傅后，病势略有起色，三个月后又突然恶化。在这几个月里，类思臥病在床，每夜从床上爬起来，跪在苦像前默祷。


耶稣圣体瞻礼后第八日到了，省会长来探访。类思对他说：「神父，我要动身了。」省会长问他到哪里去。类思道：「到天国去。」省会长对左右的人说道：「你们看，类思讲起到天国去，他说话的口气，就如我们到城里去一样。」到了晚上，类思请贝勒明神父念善终经。他眼睛注视着苦像，口里不断重复念耶稣圣名，含笑安逝，年仅二十三岁，时在一五九一年六月廿一日。


类思于一七二六年荣列圣品。教宗本笃十三世立他为青年的主保，圣人生平最令人讚叹的美德，便是弃绝世荣度玉洁冰清的生活。
