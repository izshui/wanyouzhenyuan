病人傅油及照顾病人礼典

（台湾主教团礼仪委员会编印）

 

第一章 慰问及为病人送圣体礼规

 

壹、慰问病人 

(一) 所有教友都应分担基督和教会对病人的慈爱和关怀。每位教友应尽可能访问病人，予以细心的照顾和安慰，他们有任何需要，也予以友谊的帮助。

 

(二) 堂区神父及负责照顾病人者, 尤应以信德的言词给病人解释身体疾病在救恩奥迹中的意义。也要劝导他们，使他们在信德光照之下，知道与受苦的基督相结合，以祈祷圣化他们的疾病，并由祈祷中汲取忍受病苦的毅力。

他们也有责任渐渐引导病人，在可能范围内，勤办告解、勤领圣体，尤其在适当时刻领受病人傅油和临终圣体。

 

(三) 应帮助病人自己、或与亲友或侍候病人者常行祈祷。这种祈祷最好取材于圣经；或默想有助于了解人身疾病意义的耶稣言行；或由圣咏中或由其他部份选取合适的祷词及思想。为使病人善行祈祷，必须给予适当的辅导；神父有时应乐于与病人一齐祈祷。

 

(四) 神父在访问病人时，可举行简短的圣道礼 ( 圣经诵读 )，先以友爱的交谈预作准备, 共同选取适当的经文。诵读圣经以后，最好加念一端祷词，祷词可取自圣咏，或取用其他祷词或祷文；最后可覆手祝福病人。

 

圣经诵读 ( 与病人一起举行 )

 

1. 导言 ( 可用以下言词开始 )

耶稣说过：「你们受苦的，和负重担的人，都到我跟前来！我要使你们获得休息。」现在， 我们一齐投靠我们的主耶稣，他是我们忧苦时的安慰，是我们病弱中的力量。因父及子及圣神之名。阿们。

 

2. 祷文

领：主耶稣！你的仆人病了，求你开恩，求你照顾。

答：求主俯听我们。

领：主耶稣！只要你说一句话，你的仆人就会痊愈，求你赐给你这位患病的仆人坚忍的精神。也求你说一声：「痊愈吧！」( 答 )

领：主耶稣！「你承担了我们的脆弱，背负了我们的疾病。」求你赐给你的仆人忍受病苦的力量。( 答 )

领：主耶稣！你在世时，许多人带着瘸子、残废、瞎子、哑巴和各种病人到你跟前来，你都治好了他们。求你也治好你的仆人。( 答 )

领：主耶稣！你在痛苦中曾给我们立了祈祷的榜样：「父啊！如果可能，就求你撤回我这苦杯吧！但不要按照我的意愿，而要按照你的圣意。」求你赏赐我们认识你的旨意，使我们谦恭顺从。( 答 )

 

3. 读 经 

( 可视对象选读适当的章节。见本书第七章：圣经选读 )

恭读圣玛窦福音 ( 8 : 4 - 10, 13)

那时候，耶稣进了葛法翁，有一位百夫长来到祂跟前，求祂说：「主啊！我的仆人瘫痪了，躺在家里，非常痛苦。」耶稣对他说：「我去治好他。」百夫长答说： 「主啊！你到舍下来，我不敢当，只要你说一句话，我的仆人就会好的，因为我虽是属人权下的人，也有士兵属我权下；我对这个说：你去，他就去；对另一个说： 你来，他就来；对我的仆人说：「你做这个！」他就做」耶稣听了，非常诧异，对跟随的人说：「我实在告诉你们：在以色列我从来没有见过一个人，有这样大的信心。」然后对百夫长说：「你回去，就照你所信的，给你成就吧！」仆人就在那时刻痊愈了。

( 读经后可默祷片刻；答唱咏可自第七章 183 - 196中选择 )

 

4. 答唱咏 ( 参阅咏 ：103 )

领：我的灵魂，请赞颂上主！我的灵魂，请赞颂祂的圣名，不要忘记祂的恩宠。

答：我的灵魂，请赞颂上主！

领：是祂赦免了你的各种罪过；是祂治愈你的一切痛苦；是祂解救你的生命脱离危险；是祂的温良慈善，保你平安。

答：我的灵魂，请赞颂上主！

领：祂对我们的慈爱，远比天高地厚；祂饶恕我们的罪过，永远不再追究。

答：我的灵魂，请赞颂上主！

领：祂怜爱我们，有如慈父怜爱自己的子女；祂原知道我们的来历，也知道我们不过是尘土而已。

答：我的灵魂，请赞颂上主！

领：人的岁月有如青草，又像田野盛开的鲜花，狂风一吹过，生命就会结束，再也没有人会认识他。

答：我的灵魂，请赞颂上主！

领：但上主对敬畏祂的人永远广施慈恩，并以正义对待他们的子孙。

答：我的灵魂，请赞颂上主！

( 以下经文与病人同念 )

 

5. 病中祷词

全能的天主！你创造了我，又抚育保佑我，自始至终，无时间断。我今抱病在身，深感软弱无力，疼痛难忍；求您赐给我力量，效法古圣约伯，安心忍受一切病苦。

仁慈的天父！你为拯救我们，派遣了你的圣子降生成人；恳求你使我常常想到耶稣如何代人赎罪，历经忧患，使我效法祂受苦忍辱的榜样。我不急于祈求你免除我的病苦，但求使我能仰合你的旨意，勇敢忍受。我愿意奉献我的忧苦，赔偿我所犯的罪过，我愿意接受你所赐给我的一切：无论喜乐或忧苦，健康或疾病，我都乐意接受，但求你坚强我的信心，使我常常依靠你的慈爱，总不失望悲伤。以上所求是靠我们的主基督。阿们。

 

6. 天主经

领：天主是我们的慈父，在任何境遇中，祂都照顾我们。现在我们用主耶稣教给我们的祷词，向祂说：

众：我们的天父，愿你的名受显扬，愿你的国来临；愿你的旨意奉行在人间，如同在天上。求你今天赏给我们日用的食粮；求你宽恕我们的罪过，如同我们宽恕别人一样；不要让我们陷于诱惑，但救我们免于凶恶。阿们。

 

7. 祝福 

愿全能的天主降福我们，在一切灾祸中保佑我们，引导我们到达永生。因父及子及圣神之名。阿们。

 

贰、为病人送圣体 

应注意事项

(一) 照顾人灵的牧者，应特别关注病人及年迈者，虽然他们病势并不严重、没有生命危险，也宜多次，甚至每天给他们送圣体，尤其在复活期内。送圣体可在一天中的任何时刻。

领圣体的病人不能吞咽饼形时，可只领圣血。但应遵守规定的条件 ( 见 87 页第 95 条 ) 。侍候病人者可与病人一起领圣体，但当遵守法定的规则。

(二) 为给圣堂以外的人送圣体，所携带的圣体应装在密盖的圣体盒内；送圣体者的服装和举止要合乎当地的习惯。

(三) 负责照顾病人者应整理病人房间，特别准备一张桌子，上覆白布，以便安置圣体，依照习惯，桌上也备置苦像、圣水壶、洒水具，以及蜡烛。

 

一、通常仪式

1. 致 候 

( 神父身穿适合这圣职的服装，到了病人处，和善地向病人及周围的人致候，可用以下致候词：)

 

祝你们全家平安！ ---- 答：祝神父平安！

或

愿主的平安常与你们同在。 ---- 答：也与你的心灵同在。

 

(或用其他弥撒中所用的致候词)

(神父将圣体供于桌上，与在场众人向圣体致敬)

 

2. 洒 圣 水 

(然后视情形可向病人及病房洒圣水，同时念下列经文：)

 

愿这圣水使我们想起曾领受的洗礼，并怀念以苦难和复活救赎我们的主基督。

 

3. 告解或忏悔  

(如有必要，神父先听病人告解，如病人不办告解，或有其他人也领圣体，则神父请病人及参礼者念忏悔词：)

 

弟兄们，现在请大家认罪，好能善领圣体。

 

(静默片刻，然后大家念忏悔词：)

全体：我向全能的天主和各位教友，承认我思、言、行为上的过失。(捶胸) 我罪，我罪，我的重罪。为此，恳请终身童贞圣母玛利亚、天使、圣人、和你们各位教友，为我祈求上主，我们的天主。

 

神父 ：愿全能的天主垂怜我们，赦免我们的罪，使我们得到永生。

全体：阿们。

 

4. 读圣经  

(如认为适当，可由一位参礼者，或由神父自己诵读一段圣经，譬如：)

( 一) 若 6 : 54 - 59

主说：谁吃我的肉，喝我的血，必得永生，在末日我还要使他复活；因为我的肉是真实的食物，我的血是真实的饮料。谁吃我的肉，喝我的血，他便住在我内，我也住在他内。正如那生活的父派遣了我，我因父而生活；照样，那吃我的人，也要因我而生活。这就是从天上降下来的食粮，不像你们的祖先吃了玛纳仍然死了；谁吃这食粮，必要活到永远。

( 二 ) 若 14 : 6

主说：我是道路、真理、生命，除非经过我，谁也不能到父那里去。

( 三 ) 若 14 : 23

主说：爱我的人，必遵守我的话，我父也必爱他，我们要到他那里去，并要在他那里作为我们的住所。

或其他：若 15 : 4；格前 11 : 26，若 14 : 27；若 15 : 5 ；若一 4 : 16。

( 读经后可予以简短的解释 )

 

5. 天主经  

(神父可用下列或其他类似的言词领念「天主经」：)

 

请大家祈求天主，一齐念主耶稣基督亲自教给我们的经文：

全体：我们的天父，愿你的名受显扬；愿你的国来临；愿你的旨意奉行在人间，如同在天上。求你今天赐给我们日用的食粮；求你宽恕我们的罪过，如同我们宽恕别人一样；不要让我们陷于诱惑，但救我们免于凶恶。阿们。

 

6. 送圣体

(神父举起圣体念：)

 

请看！天主的羔羊，请看！除免世罪者。蒙召来赴圣宴的人是有福的。

 

(病人及其他领圣体者同念：)

 

主，我当不起你到我心里来，只要你说一句话，我的灵魂就会痊愈。

 

(神父在病人前举示圣体说 : )

 

基督圣体。( 或：基督圣血 ) 。

病人答：阿们。

 

(然后恭领圣体，其他参礼者若要领圣体，亦照常礼恭领)

 

7. 结束祷词  

(送圣体完毕，神父照常规洗圣器，然后可默祷片刻，继续念以下结束祷词：)

 

请大家祈祷 : 主、圣父、全能永生的天主！我们的弟兄( 姊妹 ) 领受了你的圣子、我们的主耶稣基督的至圣圣体 ( 圣血 ) ，我们恳求你，使他神形受益，获得永生。以上所求是靠我们的主基督。

答：阿们。

 

8. 祝福

(然后降福病人及参礼信众 : 或以尚存有圣体的圣盒降福，或用为病人特定的祝福经文，或用弥撒后的降福经文均可。)

(第一式)

 

愿天主圣父降福你。 答：阿们。 

愿天主圣子治愈你。 答：阿们。

愿天主圣神光照你。 各：阿们。

保护你的身体，并拯救你的灵魂。答：阿们。

照顾你的身体，引领你到达永生。答：阿们。

愿全能的天主圣父、圣子、圣神也降福在场的诸位亲友。 答：阿们。

 

(第二式)

 

愿全能的天主圣父、圣子、圣神降福你们，常与你们同在。 答：阿们。

 

二、为病人送圣体简式

应注意事项

(一) 给住在同一医院不同病房的病人送圣体时，可用这简短仪式，也可加入通常仪式中的几条主要项目。

(二) 若病人愿告解，神父在送圣体前，先听告解，赦他们的罪。

(三) 仪式可在圣堂、或小圣堂、或第一个病房即行开始。

 

1. 开始礼 

(神父开始时可念以下对经：)

 

在这圣筵中，我们恭领基督，纪念祂的苦难，心灵充满恩宠，并获得来世光荣的保证。

 

2. 送圣体  

(神父由持烛者陪同，到病人前，或向同一病房的多数病人念：)

 

请看，天主的羔羊！请看，除免世罪者！蒙召来赴圣宴的人是有福的。

 

(领圣体者答念：)

 

主，我当不起你到我心里来，只要你说一句话，我的灵魂就会痊愈。

 

3. 结束祷词  

(结束祷词可在圣堂，或小圣堂，或第一个病房里念，省略降福。)

 

天主！你藉你惟一圣子的逾越奥迹，完成了人类的救赎工程；求你垂怜，使我们藉着圣事的标记，能忠诚地传报基督的圣死与复活，使你的救恩在我们身上不断地增长。以上所求是靠我们的主基督。阿们。

病人傅油
