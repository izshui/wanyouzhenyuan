真福查瓦拉
Bd. Kuriakose Elias Chavara
 1月3日

    库里亚科·查瓦拉1805年2月8日生于印度喀拉拉凯邦南卡里，排行第三，他的父母是锡罗马拉巴圣堂的虔诚教徒。他1829年晋铎，立即加入多默神父的团体，他是在这个团体接受的教育。多默神父创立新的修会，即现在的无玷圣母加尔默罗会，多默神父去世后，查瓦拉接受了这一新修会。1855年该修会获批的时候，他取了库里亚科·厄利亚斯的名字。

    这个团体的第一所会院是在马纳玛开办的。查瓦拉生前创办了五所会院，都在喀拉拉邦其它地方。他在许多领域都是一个开拓者，他创立了学校、修院、养老院，培训传教员，为司铎和教友介绍年退省处所。他为了创办一所印刷厂，出版灵修、要理书，竭尽全力，他还出版了日课经本、圣教日历和弥撒礼仪书，对统一礼仪起了很大的作用。他给跟随他的人留下了积极创新的精神。

    为了同加尔默罗传教会的白加禄神父合作，查瓦拉神父成立了一个修女院，即现在的加尔默罗母亲会，还成立了一个德肋撒加尔默罗会。他心怀喀拉拉邦全体教会，包括叙利亚和拉丁教会。

    1861年，查尔丁宗主教派多默·洛可主教向他通报玛拉巴尔叙利亚教会的情况。他玩弄权术，冒充拥有教宗特权。查瓦拉神父早已委任为委拉保里宗座代牧，因而抵制了洛可的企图，在争取各堂区反对洛可的局面里起了决定的作用。洛可终于于1862年返回原地。

    查瓦拉神父过着虔诚的祈祷生活，而且特别致力于敬礼圣母。他常常看顾病人，甚至传染病人也不例外，尤其关心疼爱穷困的人。1868年2月，他给一位本堂神父的信中说：“我们应该彼此相爱，彼此宽恕错误和缺点，这样你便可以享受现世的平安和来世的永福。不要注目你一生中那些辉煌的日子，也许就是这些日子将从你的生命中勾去。”

    他于1871年1月3日逝世，葬于古那麻马。1883年他的遗体迁葬于马纳玛，这儿便成了朝圣地。1988年2月8日，教宗若望保禄二世对印度进行牧灵巡访的时候，在喀拉拉邦的可达亚立他为真福品。

（译自《罗马观察家》报）
