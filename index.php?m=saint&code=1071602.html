圣女玛利亚•玛达肋纳•巴斯德
St. Mary Magdalem Postel
 7月16日

圣女玛利亚玛达肋纳巴斯德生于1756年11月28日,自幼热心敬主，圣德出众。当她在本笃女修会读书的时候，立志献身事主，发了终身守贞的誓愿。年18岁，学业完成，回故乡创立了一所学校，招收女生。

五年后，政府禁止司铎执行神职，圣堂被封闭。巴斯德在家里布置了一间小圣堂，拉马西神父常到这堂里秘密举行弥撒。这堂里常有圣体供奉，后来因为风声太紧，官方常来查辑，依照一般教难时期的特别通融办法，圣体由巴斯德保管，她可以随时带往各地，送给临终病人。

巴斯德长时期担任这项秘密而危险的神圣工作。她的房屋，常有军警来查访。万一被他们查出任何可疑之点，就有入狱丧命的危险，非有极大的勇气和圣德，决不能执行这样艰巨的工作。

教难平息，巴斯德开始公开传教。她招收学童，从事教育训练儿童，给儿童和成人讲解要理，举办各项慈善事业。

塞堡城学校缺乏，儿童失学的很多。巴斯德自告奋勇，筹划在该城创办学校。她身无分文，有的只是一股为人服务的毅力。她去见塞堡的加巴神父，表示愿意举办学校，让塞堡失学的儿童有受教育的机会，有认识天主的机会。她也愿意举办社会救济事业解决贫人的困难。为了实现这项目标，她决定创设一座女修院。加巴神父很赞成巴斯德的计划，租赁了一间房屋，招收学童。有3位女子，也愿意献身事主，与巴斯德合作。这间雏型的学校奉圣母为主保，取名“仁慈之母”。

1807年，巴斯德和其余３位修女正式发圣愿。

3年后，学生人数增至２百。学习科目，除了一般学校科目外，还有实用的手工技术。此外，修会每年捐募大批款项，举办慈善救济事业。

巴斯德等塞堡修院初具规模，就到别的地方创建分院，收容孤儿。她甘愿作“耕耘”的工作，而不愿分享“收获”的丰果。

巴斯德到了华洛日，城里已有三座修院，负责教育。巴斯德和她的同伴招收了一批孤儿，她们每天做手工，维持孤儿的生活。加巴神父怕巴斯德巧妇难为无米之炊，劝她放弃这计划。巴斯德很有信心地答道：“天主既然把这些女孩子给我教养，祂一定不会把他们弃置不顾。只要我一天活在世上，只要我一天有力气工作，我决不会抛弃这女孩中的任何一个。我深信天主一定会帮助我的。” 巴斯德的毅力是惊人的。那时候，她已是一位年近花甲的老妇。可是她每天工作，什么工作都做，从缝纫女工，至种田耕耘，她样样都做。两年后，有一个慈善家送了一所房屋给她，正式开办学校。

巴斯德的教育事业，逐渐发展。她六十岁那年，政府颁布法令，学校校长均须审查学历资格，经过特别考试。巴斯德老当益壮，也去参加考试，成绩及格，取得正式校长的资格。

女修会在沙物添设分院，招收了10位望修女。其中一位后来荣列真福品：福女波拉西达味霭。

1837年，修会改订会规，采用教廷批准的天主教学校男修会的会规，会务欣欣向荣。

 

巴斯德于1846年7月16日安逝，享寿90岁。死后显有灵迹甚多。1925年，这位为“天主教学校女修会”工作了41年的伟大女子，荣列圣品。
