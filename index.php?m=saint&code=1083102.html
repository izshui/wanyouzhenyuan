真福儒万纳•安西诺
沙吕查主教 Bd. Juvenal Ancina
 8月31日

1545年10月19日，儒万纳·安西诺生于西班牙福沙诺城。儒万纳自幼非常热心，但是他无意修道，父亲希望他将来成为有名医生，送他入孟波利大学攻读医科。儒万纳由孟波利转往帕雕亚大学，年24岁，在都林考取医学博士和哲学博士学位。1569年起，在都林任医学教授，同时兼设诊所，免费诊治贫人，不收分文。闲时唯一的消遣是下棋、写拉丁诗和意大利诗。诗的题材都是国家和教会重要的事，例如1572年，教宗圣比约五世驾崩，他就写了一首长诗吊唁。

有一天，他在堂里参与弥撒，听到一首描写生命短促的圣歌，顿感世事虚幻。他觉得过去的生活，不犯罪不作恶，那是不够的，今后必须积极修务圣德。儒万纳开始作虔诚的默想祈祷，养成超脱世俗的习惯。

1575年，儒万纳赴罗马，他住的房屋，离监狱、医院、贫民窟、童犯改造所很近，因为这些不幸的人都是他服务的对象。他开始研究神学，拜圣伯肋明为师，和圣斐理伯·内利交好，和罗马知名热心教徒来往。他度了三年这样的生活。他很希望入修会，但不知道哪一个修会对他最适合。

儒万纳领受了小四品的神职，在圣斐理伯·内利的指导下修务圣德。他听人说罗马一位有名的律师弃俗修道，入加多森苦修会，不禁跃跃欲试，也想学他的榜样。儒万纳的弟弟若望·玛窦也有意入加多森会，二人征询圣斐理伯·内利的意见，斐理伯·内利劝他们入经堂修会，儒万纳接受斐理伯的劝告，1578年10月1日，和他的弟弟二人同入经堂修会。

儒万纳在经堂修会修道四年后，晋升铎品，1586年，派往那不勒斯经堂修会分院服务。他在那不勒斯讲道训人，成绩非常美满，人们给他取了“霹雳之子”的外号。

那不勒斯有个有名的女歌唱家，她听了儒万纳的讲道，非常感动，立志以后不再唱流行歌曲，改唱圣歌。

儒万纳很爱好音乐，他希望晚祷经用圣乐伴奏。他也用普通乐谱，编撰圣歌。

儒万纳全力援助病人：他将那不勒斯的热心女教徒组织了一个善会，为重病人服务。他和城里一个理发师约定，他送去的穷人，理发不收钱，1595年，儒万纳在那不勒斯工作已有10年，他渴望退隐和外界隔绝，度静修的生活。但是修会当局将他调回罗马修院担任要职，儒万纳奉命前往罗马。

一年后，三个教区的主教出缺，儒万纳呼声最高，他逃往城外暂避，但修会当局命他重返罗马供职。在那一段时期，他工作的中心地点是毕爱蒙。

1602年，撒伏约有两个教区的主教出缺。教宗亲自下令命儒万纳任撒伏约沙吕查区主教。儒万纳不敢拒绝，同年9月1日祝圣为主教。

儒万纳接任后，第一件事是推行“四十小时”的敬礼。1603年，巡视本区各地，考察教务，一路上显了许多灵迹。

儒万纳回到沙吕查，指摘一个人品行不端。1604年，圣伯尔纳铎瞻礼，他出门赴宴，那人在菜里下毒，儒万纳中毒，8月31日逝世。一位修士说：“儒万纳的死，堪与圣若翰先后媲美。”

儒万纳死后显有灵迹甚多。1624年，教廷进行审查他的列品案。1869年，儒万纳荣列真福品。
