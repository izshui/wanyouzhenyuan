《病人傅油礼典》（4）
公教会礼仪发凡   2021-11-24   5035

第四章  临终圣体

 

应注意事项

 

(一) 本堂神父，和负责只照顾病人的神父，应注意使病人弥留时能领受临终圣体，因此对于病人和他的家人以及服事他的人，应在适当的时间给以牧灵的准备；但要顾及实际的情形和病人的需要。

(二) 经教会教长同意，在病人家里 ( 或病房 ) 举行弥撒时，可在弥撒中或弥撒外给病人送临终圣体，唯均须依照下面的规定。

(三) 病人不能领饼形圣体时，可仅领圣血。如果弥撒不在病人处举行，可在弥撒中保留一部分圣血，盖上圣爵，置入圣龛中；将圣血送给病人时，容器必须密封，以防流溢。给病人送圣体时，要看个别情形选用适当的方式，通常可给病人送圣体兼送圣血。送圣体后，如有圣血余剩，则由神父领之，然后洗净圣器。

(四) 参礼的教友都能领圣体兼领圣血。

 

壹、弥撒内送临终圣体

 

(五) 送临终圣体所举行的弥撒，可用「临终圣体特殊礼仪弥撒」 ( 见第三章：病人傅油弥撒，有特殊祷词 ) ，或用敬礼圣体弥撒，但遇到将临期、四旬期及复活期的主日、节日、圣灰礼仪以及圣周时，则应用当天的弥撒。礼成时可用专有的祝福式，或当天弥撒内特别祝福式。读经可从「弥撒圣经选读」中，或本礼典第七章所列之读经中选取。为了病人及参礼者的益处，亦可另行选择。

不能举行任选弥撒时，也可从上述的读经中选取一篇；但遇到圣周三日庆典、圣诞节、主显节、耶稣升天节、圣神降临、圣三节、圣体圣血节以及其他法定的庆节时，所有读经应为当日弥撒读经。 

(六) 如有必要，神父应在弥撒前听病人告解。 

(七) 弥撒照常规举行，但神父要注意以下各点：

( 1 ) 福音后，如环境许可，最好就所读圣经，对病人及参礼者作简短讲道，说明临终圣体的重要和意义。

( 2 ) 讲道后，如可能，神父请病人重发圣洗誓愿 ( 见以下贰、弥撒外送临终圣体礼，第 7号 ) ，代替弥撒中的信经。

( 3 ) 要用一适当的信友祷词 ( 可用以下贰、弥撒外送临终圣体礼第 8号的祷文 ) ，但假如已先发了圣洗誓愿，为避免病人过于疲劳，则可省略。

( 4 ) 弥撒中，神父与参礼者可依照常规给病人行平安礼。

( 5 ) 病人和参礼者均能领圣体兼领圣血。但给病人送临终圣体时，当念专有的经文 ( 见第三章：病人傅油弥撒 ) 。

弥撒后，神父可用特别的祝福词祝福病人，并可加念颁发临终大赦经文 ( 见病人傅油弥撒 ) 。

 

贰、弥撒外送临终圣体仪式

 

应注意事项

(一) 病人要告解时 ( 神父应注意提示 ) ，可在送临终圣体以前行之。若在礼仪中告解，则在礼仪开始时行之。若礼仪中不办告解，而有其他在场者要领圣体，则宜念忏悔词。

(二) 神父应穿适合此圣职的服装，除圣体外，也携带圣水。

(三) 病人处应准备一张桌子，上铺白布，置苦像，点燃蜡烛，放洗手用水及手巾。

 

1. 致候

( 神父到病人处，先和善地向病人及在场者致候。可用以下或类似的话：)

愿你们全家平安！

或：

愿主的平安常与你们同在。

( 神父把圣体供于桌上，与在场众人一同致敬。)

 

2. 洒圣水 

( 神父可向病人及病房洒圣水，或让病人蘸圣水划十字圣号，同时念以下经文：)

愿这圣水使我们想起曾领受的洗礼，并怀念以苦难和复活救赎我们的基督。

 

3. 导言

( 然后用以下，或其他更适合病人情况的言词，向在场者致词 : )

各位教友：主耶稣基督经由这世界回到父那里去以前，就是在受难的前夕，给我们留下了自己的圣体圣血，使我们度过今生回到他那里去的时候，饱飨圣体圣血的食粮，作为复活的凭证。现在我们本着爱德与生病的弟兄／姊妹聚在一起，请大家为他祈祷。

 

4. 忏悔

( 如需要，神父先听病人的告解：紧急情形下，如不能详细告明，只告明罪类即可。如病人不办告解，而有其他领圣体的人，则神父请病人及参礼者念忏悔词：)

神父：各位教友：现在请大家认罪，好能善领基督圣体。

( 省察片刻, 然后大家念忏悔词 )

全体：我向全能的天主和各位教友，承认我思、言、行为上的过失。( 捶胸 ) 我罪、我罪、我的重罪。为此，恳请终身童贞圣母玛利亚、天使、圣人、和你们各位教友，为我祈求上主、我们的天主。

神父：愿全能的天主垂怜我们，赦免我们的罪，使我们得到永生。( 答 : 阿们。)

 

5. 临终大赦 

(病人办告解或发痛悔后，在此生命末刻，神父可颁给临终大赦：)

我现在以宗座所授之权，给你全大赦，并赦免你的一切罪过。因父及子及圣神之名。(答：阿们 )

或 :

愿全能的天主，因他救赎人类的神圣奥迹，赦免你今生来世的罪罚，开启天堂之门，领你进入永福的天乡。( 答：阿们 )

 

6. 读经

( 由一位参礼者或神父自己读一段圣经，如：)

(1) 耶稣说：「谁吃我的肉并喝我的血，必得永生，在末日我要叫他们复活：因为我的肉是真实的食粮，我的血是真实的饮料」( 若 6 : 54 - 55 )

(2) 耶稣说：「我是道路、真理和生命，除非经过我，谁也不能到父那里去」( 若 14 : 6 )

(3) 耶稣说：「谁爱我，必遵守我的话，我父也必爱他，我们要到父那里去，以他那里作我们的住所。」( 若 14 : 23 )

或其他合适的章节，如：若 6 : 54 - 59 ; 15 : 4 ; 格前 11 : 26 。

 

7. 领洗誓愿

(如病人的情况许可，应使病人重发领洗誓愿 )

神父：你信全能的天主圣父创造天地万物吗？

病人：信。

神父：你信我们的主耶稣基督、天主的唯一子，由童贞玛利亚降生，受苦受难，死而复活，现今在天上享受光荣吗？

病人：信。

神父：你信圣神，圣而公教会，诸圣的相通，罪过的赦免，肉身的复活，和永恒的生命吗？

病人：信。

 

8. 祷文

( 如病人情况许可，可念简短祷文，请病人尽可能与在场者作答：)

神父：各位教友，现在我们向耶稣基督，众口一心地共同祈祷。

领：主耶稣爱我们，为使我们得到生命，他竟然牺牲了自己的生命。现在请为我们这位弟兄／姊妹祈祷，求主恩赐他慷慨地奉献自己的生命。我们同声祈祷。

答：求主俯听我们。

领：主耶稣曾经说过：谁吃我的肉，喝我的血，必得永生。现在请为我们这位弟兄／姊妹祈祷，求主藉自己的身体，赐给他永生的保证。我们同声祈祷。

答：求主俯听我们。

领：主耶稣邀请我们共赴天国的圣宴，在那里再没有痛苦、悲伤、忧愁与别离。现在请为我们这位弟兄／姊妹祈祷，求主恩赐他与天上神圣欢聚一堂，共享永福。我们同声祈祷。

答：求主俯听我们。

 

9. 临终圣体  

( 神父可用下列或其他类似的言词领念「天主经」：)

请大家祈求天主，一齐念主耶稣亲自教给我们的经文：

全体：我们的天父，愿你的名受显扬；愿你的国来临；愿你的旨意奉行在人间，如同在天上。求你今天赏给我们日用的食粮；求你宽恕我们的罪过，如同我们宽恕别人一样；不要让我们陷于诱惑，但救我们免于凶恶。阿们。

 

( 神父举起圣体念：)

请看，天主的羔羊！请看，除免世罪者！蒙召来赴圣宴的人，是有福的。

(病人及其他领圣体者同念 : )

主！我当不起你到我心里来，只要你说一句话，我的灵魂就会痊愈。

( 神父到病人前举起圣体说：)

基督圣体( 或 : 基督圣血 )

( 病人答：阿们 )

( 神父送圣体后立即说：)

愿他保护你，引领你到达永生。

病人答：阿们

( 送圣体后，神父照常规洗涤圣器，默祷片刻 )

 

10. 礼成式

( 神父念以下祷词 : )

请大家祈祷 : 天主！你的圣子是我们的道路、真理和生命；求你以无限仁慈眷顾你的仆人（某），使他赖你的许诺，领了你圣子的圣体 ( 圣血 ) 之后，能平安地荣登天国。以上所求是靠我们的主基督。

众答：阿们。

( 然后祝福病人及参礼者 : )

愿全能的天主圣父、圣子、圣神降福你们。

众答 : 阿们。

病人傅油
