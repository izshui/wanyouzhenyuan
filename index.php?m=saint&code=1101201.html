圣威弗烈
约克主教 St. Wilfrid 709
 10月12日

圣威弗烈是初期英国教会最著名神职人士之一。公元634年出生，幼年丧母，继母常常虐待他。所以十三岁那年，他就离家到诺森保利亚王奥斯维的宫廷作事。依弗兰王后送他到林弟法修院学校读书，研究圣经学。学业完成，威弗烈到法国、意大利考察修院制度，在坎特伯里住了个时期，从圣和诺略学习罗马修院制度。

654年，圣皮斯哥启程赴罗马，经根德，威弗烈就随他一同出发。威弗烈在里昂逗留了一年，里昂主教圣亚纳门很赏识他的才能，留他久住，威弗烈不肯。到了罗马，拜波尼法爵为师。波尼法爵博学多才，他是教宗圣玛尔定的秘书。威弗烈在罗马住了三年，行了剪发礼后，取道返国。国王知道他对罗马教会制度很有研究，请他在国内逐步实施，籍以代替过去的塞尔特制度。国王在里本创设了一座修院，威弗烈任修院院长，采用圣本笃会的会规，不久，他领受司铎神品，主持祝圣礼的是西撒克逊主教圣亚奇伯。

威弗烈以全力劝导当地教会采用罗马制度。那时候，最大的问题，是复活瞻礼日期的计算。由于塞尔特式计算方法与罗马制颇有出入，复活瞻礼的日期，就有差别。大家认为必须采用罗马统一瞻礼日期。公元663年（或664年），特在史太诺召集会议，出席会议的人大多数赞成采用罗马制。

威弗烈赴法国，由圣亚奇伯祝圣为主教。669年起，任约克主教。他每次到各地视察教务，都是安步当车，热心传扬教务，可国王与他不睦。威弗烈的教区被坎特伯里总主教划分为三部分，威弗烈不服，向罗马教廷申诉，请教宗主持公道。这是英国教会历史上第一次向罗马申诉的例子。

威弗烈赴罗马途中，因风势不顺，船只在菲士兰靠岸。威弗烈就在该地传教，劝化异教徒无数。679年，他到了罗马，觐见教宗圣亚加多。教宗召集会议，决定威弗烈恢复原职，教区不应划分。威弗烈带了教宗的诏书，回到英国。国王大怒，将他羁押监狱九个月后获释。

威弗烈到南撒克逊，南撒克逊国王是一个教友，对威弗烈表示热烈欢迎。南撒克逊人民，全部皈依圣教。威弗烈到南撒克逊的时候，农禾干枯，就在他第一次举行洗礼的那一天，甘霖沛然而降，农民额手称庆。威弗烈在撒西创建修院，主教府公署亦设该地。

圣威弗烈在撒西住了五年。圣刁多禄病重，请威弗烈到伦敦相会；圣刁多禄表示希望威弗烈将来接他的位，继任坎特伯里总主教。威弗烈道：“未来坎特伯里主教的人选，应召开会议决定。但是我原任约克主教，被国王驱逐，请你设法调停，命名我恢复原职。”刁多禄写信给国王及教会首长，威弗烈重返旧任。五年后，国王与威弗烈发生冲突，威弗烈再度被逐，

703年，威弗烈被撤职，他不服，往罗马向教廷申诉。

教宗召集会议，宣布威弗烈应维持原职。威弗烈为了表示让步起见，自动将约克区教务划交圣若望皮弗利管理，自己仅保留海兴教区。威弗烈将主教府所有财产，分给各修院教堂，709年逝世。

威弗烈一生，屡遭波折。可是他的圣德，有口皆碑，就是反对他的人，也承认他是一位才德双全的主教。
