圣女玛利亚•罗撒
童贞伯来西亚仁爱婢女会创办人 St. Mary di Rosa 1813-1855
 12月15日

18世纪初叶，伯来西亚出了三位圣德卓越的人物：真福鲁道维·巴华尼、福女德肋撒、圣女玛利亚·罗撒。

玛利亚·罗撒生于1813年，父亲名叫克肋孟，兄弟姊妹有九人之多，圣女行六。年十一岁，慈母离世；十七岁辍学，治理家务。

克肋孟拟代女儿物色如意郎君，玛利亚大惊，和神师福斯帝诺神父商量。福神父亲自去见克肋孟，劝他尊重女儿守贞的志愿。克肋孟很开通，表示绝不强迫玛利亚结婚。

玛利亚在家住了十年，创办社会福利事业。克肋孟有一个织布厂和一个农场，雇有女工多名，玛利亚照顾她们灵魂的需求。她和本堂神父合作，组织了一个女工善会，举办退省善工。

1836年，伯来西亚发生大疫，玛利亚征得父亲同意，往疫区医院看护病人。

社会热心人士创办了一个工厂，招雇无家可归的贫苦女工。那时玛利亚24岁，被邀担任工厂管理员。两年后，玛利亚创办了宿舍和聋哑女子学校各一所。学校创建就绪，玛利亚将它移交给嘉诺撒女修会接办。

以一个年未满三十岁的少女，一手创办这些事业，确是难能可贵的。玛利亚不但能干，她的学问也很好，精通神学。

l840年，玛利亚着手组织一座女修会。与她合作的是巴纳弟夫人。玛利亚由宾叔尼神父指定为修女会会长，修会以照管在医院留医的病人为目的，定名为仁爱婢女会。最初只有会员四人，住在医院附近的一所破旧不堪的房屋里，不久，修女人数增至三十二人。

仁爱婢女会的工作，博得社会各界人士的赞誉。伯来西亚有一个医生在报上刊载专文介绍修女的服务精神。修会成立仅三日，克利马纳的居民，邀请玛利亚派修女往该地工作，玛利亚欣然接受这项邀请。

玛利亚的父亲克肋孟送了一所房屋给仁爱婢女会充会舍。1843年，主教批准了修会的临时会规。

数月后，巴纳弟夫人逝世，这对于仁爱婢女会是一个很大的打击，玛利亚丧失了一位得力的助手。可是她遇有疑难的问题，还可以向宾叔尼神父请教，修会继续扩展，服务的医院数目不断增加。

l848年，宾叔尼神父去世。是年欧洲政局动荡不安，烽烟遍地，战火蔓延到意国北部。玛利亚动员修女救护伤兵、平民、俘虏，修女们除了医疗身体的创伤外，也医疗灵魂的创伤。看护病人的身体和灵魂，这是修女和一般护士不同之处。

1849年，伯来西亚发生变乱，玛利亚和全院修女全力救护难民。一部分不守军纪的兵士拟冲入病院抢劫，玛利亚带了六名修女站在大门口，她们拿了一尊大十字架，旁边还有两名修女各持圣烛一支。兵士逡巡不前，迟疑了一会儿，纷纷作鸟兽散。这尊十字架至今还保存在伯来西亚医院里，留作纪念。

如上前述，玛利亚创办的仁爱婢女会，同时照顾病人的身体和灵魂，修女们每日一部分时间祈祷。1850年，玛利亚亲赴罗马，向教廷办理修会会规批准手续。10月24日，蒙教宗接见。两月后，会规由教廷正式批准。1852年，玛利亚和首批修女矢发圣愿。

1855年，玛利亚突患重病，她的年龄不算大，只有四十二岁，可是多年辛劳，大大削弱了她的健康。耶稣受难瞻礼左右，她的重病突然神奇地痊愈了。修会还有几件工作要她办理：那一年夏季伯来西亚的霍乱病救护工作，谭马西亚和万隆纳添设分院的工作。玛利亚办完了这几件工作，体力不支，回到伯来西亚不久，安逝主怀，时为1855年12月15日。

玛利亚一生的工作精神，可以概括在下面这几句话里．“假如我错过了一个机会，即使是最微小的机会，未能阻止别人做一件恶事，或帮助别人做一件好事，我那天晚上睡觉的时候，良心上就觉得不平安。”玛利亚一接到某人患病的消息，就去探视，帮助他安死善终。

玛利亚于1954年荣列圣品。
