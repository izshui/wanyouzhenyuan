第二章 病人傅油仪式

 

壹、通常仪式 

 

应注意事项

（一）在举行病人傅油圣事以前，神父应先了解病人的状况，以便能在准备及选读圣经上有一个目标。在可能范围内应向病人及其家人讲解圣事的意义，与他们共同准备举行此圣事。

 （二）若病人需要告解，神父尽可能在举行傅油礼以前听告解。若病人在傅油时需办告解，则可在傅油礼开始时听告解。否则，在适当时刻举行忏悔仪式。

 （三）为并非卧床不起的病人，可在圣堂或其他适当的地方，预备一个座位，坐着领受傅油圣事，使亲友们易于集合参与这个仪式。在医院同一病房中，如有其他卧床病人，神父亦应顾及他们。因为他们或许也能参与礼仪，也可能感到厌倦；如果他们不是信徒，也可能引起反感。

 （四）为多数病人傅油时，也用下面提供的仪式，覆手及傅油礼和行礼时应念的经文，应给每位病人个别举行。其他经文只用多数，向全体念一次即可。

（五）此傅油礼宜在一种安详的气氛中举行。所以地方的布置要特别注意。可视地方情形，用鲜花、蜡烛等加以点缀，以示庄严。应准备一张桌子，以便放置圣油，或要祝圣的油，以及圣体 ( 如病人行傅油礼后立刻领圣体 )。

 （六）神父在开始行礼之前，可将一些任务 ( 如读经、信友祷词等 ) 分配给参礼者，以便参礼者更积极地参与。

 

一、准备礼

1. 致候

( 神父身穿适合这圣职的服装，到了病人处，和善地向病人及周围的人致候，可用下列致候词之一：）

(一) 祝你们全家平安。

答：祝神父平安。

(二) 愿天父的慈爱，基督的圣宠，圣神的恩赐，与你们同在。( 或：愿主的平安常与你们同在。) 答：也与神父同在。

(三) 愿天父和基督，赐给你们恩宠及平安。答：赞美天父和基督。

 

2. 洒圣水 

( 神父可向病人及病房洒圣水，也可让病人及参礼者蘸圣水划十字圣号，同时念下列经文：)

愿这圣水使我们想起曾领受的洗礼，并怀念以苦难和复活救赎我们的基督。

 

3. 导言  

( 神父向全体参礼者用以下或类似的话致词说：)

各位教友：福音上记载，有许多患病的人，来请求主耶稣基督医治, 祂也曾为我们受过极大的苦难。他现在就在我们中间，因为我们此刻是因他的名义聚集在一起的，他藉圣雅格宗徒的话，嘱咐我们说：「你们中间有患病的吗？该请教会的长老们来为他祈祷，因主的名给他傅油。出自信德的祈祷，必能拯救那病人，主会使他起来；如果他犯了罪，也必获得赦免。」

因此，我们把这位生病的弟兄 ( 姊妹 ) 托付于基督的恩宠和德能护翼之下，求主减轻他的病痛，并赐他早日康复。

 

( 或念以下祷词：)

主基督！你的宗徒雅格曾说：「你们中谁有病，该请教会的长老去为他祈祷，因主的名给他傅油。出自信德的祈祷，必能拯救那病人，主会使他起来，如果他犯了罪，也必获得赦免。」

我们恳求你，助佑因你的名在此聚会的众人，并以你的仁慈保护我们这位患病的弟兄／(姊妹) (某)，以及其他患病者，使他们勇敢地忍受病苦。你是天主，永生永王 。

答：阿们。

 

4. 忏悔经  

( 如病人此时不办告解，则举行以下忏悔仪式，以准备领受傅油圣事。告解可在此时，或傅油礼以前举行 )

神父：各位教友：现在请大家认罪，使我们能虔诚地参与这神圣的礼仪。( 省察片刻 )

全体：我向全能的天主和各位救友，承认我思、言、行为上的过失。( 捶胸 ) 我罪、我罪、我的重罪。为此，恳请终身童贞圣母玛利亚、天使、圣人、和你们各位教友，为我祈求上主、我们的天主。

神父：愿全能的天主垂怜我们，赦免我们的罪，使我们得到永生。( 答 : 阿们。)

 

二、圣道礼 

( 可由一位参礼者或由神父自己诵读一段或数段圣经，神父可视病人需要选择适当的圣经章节，并加以简短的解释 )

 

5. 读经 

恭读圣玛窦福音 ( 5 : 24 - 34 )

那时候，有一大群人跟随着基督，拥挤着他。有一个妇人，患血漏已有十二年。她在许多医生手里，受了许多痛苦，花尽了自己所有的一切，不但没有见效，病势反而更加重了。她听了有关耶稣的传说，便来到人群中，从后边摸了耶稣的衣裳，因为她心里想：「我只要一摸他的衣裳，必然就会好的。」她的血漏立刻就止住了，并且觉得身上的疾病也好了。耶稣立时觉得有一种能力从他身上出去，就在人群中回过头来说：「谁摸了我的衣裳？」他的门徒向他说：「你看！群众四面拥挤着你，你还问：「谁摸了我？」」耶稣四周观看，要看做这事的妇人。那妇人明知在自己身上所成就的事，就战战兢兢地前来，跪在耶稣面前，把实情完全告诉了他。耶稣便向她说：「女儿，你的信德救了你，平安去罢！你的病已痊愈了！」

6. 祷文

弟兄们：请大家怀着坚强的信德，为我们的弟兄／姊妹(某)恳切祈祷。

主基督，你承受了我们的脆弱，负担了我们的痛苦，求你垂怜我们的弟兄／姊妹。

( 答：主基督，求你垂怜。)

主基督，你一生怜悯受苦的群众，到处给人施惠治病，求你垂怜我们的弟兄／姊妹。

( 答：主基督，求你垂怜。)

主基督，你曾命令你的宗徒，要他们在病人身上覆手，求你垂怜我们的弟兄／姊妹。

( 答：主基督，求你垂怜。)

 

7. 覆手 

( 神父在病人头上覆手，不念经文。覆手表示祈求天王赐予圣神和祂的恩宠：医治病人和祝福 )

 

8. 祝圣病人油  

( 在紧急时，如无主教祝圣的病人油，神父可用以下的经文祝福病人油：)

天主、予人安慰的慈父！你曾藉你的圣子医治人的疾苦，求你俯听我们出自信心的祈祷，从天上派遣施慰的圣神，降福这取自植物的油脂，以滋养我们的身体；愿你的降福，使所有敷用这油的人，都获得身体和灵魂的保障，除去一切痛苦、衰弱和疾病。

天主！求你使这经你圣化的油，能为我们产生治病的神效。因我们的主耶稣基督之名。 ( 答：阿们 )

( 如用已经祝圣的油，神父可念以下感恩祷词：)

慈爱的天主！求你垂顾你的仆人 (某)，他满怀着信心即将领受傅油礼；愿圣神与他常在一起永不分离；使他勇敢接受病苦，恒心祈祷，愉快地期待着主的恩许。以上所求是靠我们的主基督。( 答：阿们 )

 

9. 傅油礼

( 神父取圣油傅抹病人的额和双手，念一次：)

藉此神圣的傅油，并赖天主的无限仁慈，愿天主以圣神的恩宠助佑你； 答：阿们。赦免你的罪，拯救你，并减轻你的病苦。

答：阿们 。

 

10. 祷词

( 然后神父请参礼者一齐祈祷：)

请大家祈祷：我们的救主！求您以圣神的恩宠，医好病人的虚弱，治愈他的创伤，赦免他的罪过，消除他身心的痛苦，恢复他身心的健康，使他克尽原有的职务。您是天主 、永生水王。(答：阿们 )

--- 天主的惟一圣子，我们赞美你！你自愿降临人间，医治我们的病苦。(答：赞美天主 )

--- 施慰的天主圣神，我们赞美你！你以永恒的德能，坚强我们脆弱的肉躯。(答：赞美天主 )

上主！求你垂顾，以你的降福，圣化这为治疗你信众的病苦而准备的油，使凡敷用此油的人，藉着信德的祈祷，能解除病苦，获得力量。以上所求是靠我们的主基督。( 答：阿们 )

 

( 如用已经祝圣过的油，神父可念以下感恩祷词：)

上主！求你使敷用这圣油的信徒，能减轻他的痛苦，治愈他的疾病。以上所求是靠我们的主基督。(答：阿们)

 

9. 傅油礼

(神父用圣油傅抹病人的额和双手，念一次：)

藉此神圣的傅油，并赖天主的无限仁慈，愿天主以圣神的恩宠助佑你；(答：阿们)

赦免你的罪，拯救你，并减轻你的病苦。(答：阿们)

 

10. 祷词 

(然后神父请参礼者一齐祈祷：)

请大家祈祷 (默祷片刻) : 主，耶稣基督！你为了救赎人类，并医治我们的疾病，曾屈尊就卑而降生成人；求你垂顾你这位身心患病的仆人；我们已因你的名给他傅了油，求你使他赖你的助佑，能脱离险境，早日康复(能忍受病苦，分担你的苦难)。您是天主，永生永王。

答：阿们。  

 

四、领圣体及礼成式 

(祷词后，念天主经，然后可给病人送圣体；如不送圣体，念天主经后即以祝福式结束)

 

11. 天主经

(神父可用以下或类似的言词领念天上经：)

请大家祈求天主，一齐诵念主耶稣基督亲自教给我们的经文：

全体：我们的天父，愿你的名受显杨；愿你的国来临；愿你的旨意奉行在人间，如同在天上。求你今天赏给我们日用的食粮；求你宽恕我们的罪过，如同我们宽恕别人一样；不要让我们陷于诱惑，但救我们免于凶恶。阿们。

 

12. 领圣体  

( 神父举起圣体念：)

请看，天主的羔羊！请看，除免世罪者！蒙召来赴圣宴的人是有福的。

( 病人及其他领圣体者同念：)

主！我当不起你到我心里来，只要你说一句话，我的灵魂就会痊愈。

( 神父到病人前举起圣体说：)

基督圣体( 或 : 基督圣血 )

( 病人答：阿们 )

( 如果是临终圣体，则加念：)

愿他保护你，引领你到达永生。

( 病人答：阿们 )

( 送圣体后，神父照常规洗涤圣器 )

 

13. 结束祷词 

( 一 ) 为普通病人送圣体后念：

请大家祈祷：主、圣父，全能永生的天主！我们恳求你，使我们这位弟兄(姊妹) 领受了您圣子的圣体 (圣血) 之后，神形受益，获得永生。以上所求是靠我们的主基督。

答：阿们。

( 二 ) 为临终病人送取体后念：

请大家祈祷：天主！你的圣子是我们的道路、真理和生命，求你以无限仁慈眷顾你的仆人(某)，使他赖你的许诺，领了你圣子的圣体 (圣血)之后，能平安地进入你的天国。以上所求是靠我们的主基督。

答：阿们。

保护你的身体，并拯救你的灵魂。答：阿们。

贰、病人弥留时施行圣事程序

 

第一式 告解、傅油、临终圣体连续举行时用

应注意事项

( 一 ) 如果病人要办告解(神父应注意提示)，神父尽可能在傅油及送临终坚体以前听告解。若在行礼仪时告解，则在礼仪开始时、傅油礼以前听告解。如在礼仪中不行告解，则视情形举行忏悔式。

( 二 ) 病人在危急时，应立即行傅油礼，只在一处(额上) 傅油即可，然后立即送临终圣体。如病人已接近死亡边缘，没有充足时间时，应立即先送临终圣体，使他离开此世时，领了基督的圣体 (圣血) 而神力充沛，并拥有复活的保证。因为信友在病危时，依法应领受圣体。  

( 三 ) 病危时，坚振与傅油圣事，尽可能不要连续举行，以免两种傅油混淆了此两种不同的圣事。但如有必要，则可在祝圣病人油以前先行坚振，而省略属于病人傅油的覆手礼  

 

1. 致候  

( 神父身穿适合这圣职的服饰，到了病人处，和善地向病人及周围的人致候，可用下列致候朗：)

祝你们全家平安！

或

愿主的平安常与你们同在。 ( 答：祝神父平安 )

 

( 神父将圣体供于桌上，与在场众人向圣体致敬 )

 

2. 洒圣水 

(然后向病人及病房洒圣水，同时念下列或专用的礼仪书上所提供的经文：)

愿这圣水使我们想起领受的洗礼，并怀念以苦难和复活救赎我们的基督。

 

3. 导言 

(一) ( 如时间许可，可用亲切的交谈准备病人善领圣事，并用圣经上的话激起悔罪、爱主之情。可用以下或其他更适合病人情况的言词，向在场者致词：)

亲爱的弟兄姊妹们 : 主耶稣常在我们中间，不断地分施圣事的恩宠：他通过神父的职权，宽赦真心忏悔的人；他以病人傅油使病人坚强起来；他以自己的圣体圣血，作病人旅途上的食粮，支持病人对永生的希望。现在我们这位弟兄 ( 姊妹 ) 要求领受这三件圣事，请大家以友爱和虔诚的祈祷帮助他。

 

( 二 ) (如果不领临终圣体，则用以下导言） 

亲爱的弟兄姊妹们 : 主耶稣基督曾藉圣雅格宗徒的话，嘱咐我们说：「你们中间有患病的吗？该请教会的长老们来为他祈祷，因主的名给他傅油：出自信德的祈祷必能拯救那病人，主会使他起来；如果他犯了罪，也必获得赦免。」

因此我们将这位患病的弟兄／姊妹托付给基督的恩宠和德能，求主减轻他的病痛，并赐他获得健康。

 

4. 告解

(如需要，神父先听病人的告解，紧急情形下，如不能详细告明，只告明罪类也可。如病人不在此时办告解，则神父邀请病人及其他参礼者念忏悔词：)

神父：各位教友：现在请大家认罪，使我们能虔诚地参与这神圣的礼仪。

(静默片刻，然后大家念忏悔词：)

全体：我向全能的天主和各位救友，承认我思、言、行为上的过失。( 捶胸 ) 我罪、我罪、我的重罪。为此，恳请终身童贞圣母玛利亚、天使、圣人、和你们各位教友，为我祈求上主、我们的天主。

神父：愿全能的天主垂怜我们，赦免我们的罪，使我们得到永生。( 答 : 阿们。)

 

5. 临终大赦

(病人办告解或发痛悔后，在此生命末刻，神父可颁给临终大赦：)

我现在以宗座所授之权，给你全大赦，并赦免你的一切罪过。因父及子及圣神之名。(答：阿们 )

或 :

愿全能的天主，因他救赎人类的神圣奥迹，赦免你今生来世的罪罚，开启天堂之门，领你进入永福的天乡。( 答：阿们 )

 

6. 领洗誓愿  

(如病人的情况许可，应使病人重发领洗誓愿 )

神父：你信全能的天主圣父创造天地万物吗？

病人：信。

神父：你信我们的主耶稣基督、天主的唯一子，由童贞玛利亚降生，受苦受难，死而复活，现今在天上享受光荣吗？

病人：信。

神父：你信圣神，圣而公教会，诸圣的相通，罪过的赦免，肉身的复活，和永恒的生命吗？

病人：信。

 

7. 祷文  

( 如果病人情况许可，也可念简短祷文，使病人尽可能与在场者一起作答。这祷文可加以适应，以便更能表达病人和参礼者的祈祷 )

亲爱的弟兄姊妹们 : 主耶稣基督今天使我们这位弟兄／姊妹 (某) 领受他的圣事，现在我们一齐为他向天父祈祷。

---仁慈的天主父！求你看在受难圣子的面上，眷顾他，我们同声祈祷。答 : 求主俯听我们。

---求以你的圣爱坚强他，保护他，我们同声祈祷。答：求主俯听我们。

---求你赐他毅力及平安，我们同声祈祷。答：求主俯听我们。

 

( 如应施行坚振，可依照坚振礼规在此行之。然后省略以下第 8号的覆手礼 )

( 神父向病人头手念：)

全能的天主、我们的主耶稣基督之父！你以水及圣神使这位信友重生，赦免他的罪过；求你派遣施惠者圣神，降临到他心中，赐给他智慧和聪敏之神，超见和刚毅之神，明达和孝爱以及敬畏之神。以上所求是靠我们的主基督。(答：阿们 )

( 然后以右手拇指蘸坚振圣油，在领坚振者额上划十字念：)

O O，请藉此印记领受天恩圣神。(病人答：阿们)

(其他准备或结束礼经文，也可按照个别情形加念一些；在特别紧急的情况中，只念傅油主要经文并同时傅油即可：)

O O，请藉此印记领受天恩圣神。

 

8. 覆手

( 神父在病人头上覆手，不念经文 )

 

9. 祝圣病人油  

( 如需要祝圣病人油，神父可用以下经文祝圣：)

上主，求你降福这受造之油，和领受傅油的病人。

(如用已经祝圣过的油，神父可念以下感恩祷词：)  

--- 全能的天主圣父！我们赞美你，你为了我们，和我们的得救，派遣了你的惟一圣子来到世上。( 答：赞美天主 )

--- 天主的惟一圣子！我们赞美你，你自愿降临人间，医治我们的病苦。(答：赞美天主 )

--- 施慰的天主圣神！我们赞美你，你以永恒的德能，坚强我们脆弱的肉躯。(答：赞美天主 )

--- 上主！求你使敷用这圣油的信徙，能减轻他的痛苦，治愈他的疾病。以上所永是靠我们的主基督。

(答：阿们 )

 

10. 傅油礼  

(神父用圣油敷抹病人的额和双手，念一次：)

藉此神圣的傅油，并赖天主的无限仁慈，愿天主以圣神的恩宠助佑你；

答：阿们。

赦免你的罪，拯救你，并减轻你的痛苦。

答：阿们。

 

11. 天主经

( 神父可用以下或类似的言词领念天上经：)

请大家祈求天主，一齐诵念主耶稣基督亲自教给我们的经文：

全体：我们的天父，愿你的名受显杨；愿你的国来临；愿你的旨意奉行在人间，如同在天上。求你今天赏给我们日用的食粮；求你宽恕我们的罪过，如同我们宽恕别人一样；不要让我们陷于诱惑，但救我们免于凶恶。阿们。

 

12. 领圣体

(神父举起圣体念)

请看，天主的羔羊！请看，除免世罪者！蒙召来赴圣宴的人是有福的。

(病人及其他领圣体者同念)

主！我当不起你到我心里来，只要你说一句话，我的灵魂就会痊愈。

(神父到病人前举起圣体说)

基督圣体(或 : 基督圣血)

(病人答：阿们)

(神父然后加念)

愿他保护你，引领你到达永生。

(病人答：阿们)

(送圣体后，神父照常规洗涤圣器)

 

13 结束祷词

(一)

请大家祈祷：天主！你的圣子是我们的道路、真理和生命；求你以无限仁慈眷顾你的仆人 (某)，使他赖你的许诺，领了你圣子的圣体 (圣血) 之后，能平安地荣登天国。以上所求是因我们的主基督。

答：阿们。

 

(二 ) (如未送临终坚体则念：)

请大家祈祷：仁慈的圣父！你认识心地善良的人，乐于宽恕人的罪过，你从不拒绝向你求饶的人；求你垂怜这位弥留中的仆人(某)，使他藉傅油圣事及我们发自信德祈祷的扶助，减轻他身心的痛苦，使他获得罪赦，并因你的慈爱获得力量。因你的圣子基督，他战胜了死亡，给我们开启了永生的门路。他是天主，永生永王。

答：阿们。

 

14. 祝福 

愿全能的天主圣父、圣子、圣神，降福你们。

答：阿们。

 

第二式 不送临终圣体的傅油礼 

(如由于环境关系，病人垂危时，只能傅油，而不能送圣体时，则礼仪程序除以下两点外，可照以上第一式进行：)

 

1. 导言

(开始的导言，可用以下或类似的话取代：)

亲爱的弟兄姊妹们 : 主耶稣基督，曾藉圣雅格宗徒的话，嘱咐我们说：「你们中间有患病的吗？该请教会的长老们来为他祈祷，因主的名给他傅油：出自信德的祈祷必能拯救那病人，主会使他起来；如果他犯了罪，也必获得赦免。」

因此我们将这位患病的弟兄／姊妹托付给基督的恩宠和护翼之下，求主减轻他的病痛，并赐他获得健康。

 

2. 结束祷词

(宜选用适合病人情况的祷词)

 傅油礼后祷词

(1) 为年老者

上主！你的仆人年老患病；为了拯救他的灵魂肉身，求你藉这傅油，恩赐宠佑，惠然眷顾，使他由圣神获得充沛的力量，能保持坚强的信德和望德，给众人树立忍耐的榜样，并对你的慈爱照顾，表示出衷心的喜乐和感激。以上所求是靠我们的主基督。

(2) 为垂危病人

主耶稣基督，人类的救主！你受尽苦难，承担了我们的痛苦和脆弱。现在，我们为这位患病的弟兄／姊妹 ( 某 )。谦恭地求你，使他获得你的救赎，坚定他得救的信念，减轻他身心的痛苦。你是天主，永生永王。( 答：阿们 )

(3) 为领傅油及临终坚体的病人

天主，仁慈的圣父！你是受苦人的安慰，求你惠然眷顾这位依恃你的仆人 ( 某 )，使他的病苦，因领受傅油的恩宠而获得减轻，并使他所领的圣体 ( 圣血 )，成为他旅途上的食粮，帮助他迈向永生。以上所求是因我们的主基督。

( 答： 阿们 )

(4) 为弥留的病人

仁慈的圣父！你认识心地善良的人，乐于宽赦人的罪过，你从不拒绝向你求饶的人；求你垂怜这位弥留中的仆人 ( 某 )，使他藉傅油圣事及我们发自信德的祈祷的扶助，减轻他身心的痛苦，使他获得罪赦，并因你的慈爱获得力量。因你的圣子基督，为我们战胜了死亡，开启了永生之门，他与你永生永王。( 答 ：阿们 )

 

 

第三式 有条件的傅油 

(如果神父怀疑病人是否还活着，可照下式为他傅油 )

(如有时间，到病人前先念：)

 

现在我们满怀信心，为我们的弟兄／姊妹( 某 )祈祷，求无限仁慈的天主眷顾他，藉此神圣的傅油增加他的力量。

答：求主俯听我们。

 

(立即予以博油，用以下的经文：)

如果你还活在人间，藉此神圣的傅油，并赖天主的无限仁慈，愿天主以圣神的恩宠助佑你；赦免你的罪，拯救你，并减轻你的病苦。

答：阿们。

(然后，视情形，可加念适合病人情况的结束祷词，见第二式2号 )
