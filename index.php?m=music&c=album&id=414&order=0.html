圣若瑟修有各种圣德，思言行为各得其宜。贞洁、安贫乐道、谦卑、服从等，无不登峰造极。圣教会对他的敬礼，先抑后扬，正象征着圣若瑟谦抑的典型。我们由福音得知圣若瑟确为天主忠信的仆人；明智、忠贞、全心地完成了天主交付他的使命。尤其他的信德、服从使他成为整个人类救恩的工具。福音给我们把圣若瑟的美德描述为：“义人”及“明智的忠仆”。“义人”是指他做圣母的净配而言的；“明智的忠仆”是指被立为圣家之长和守护圣子耶稣而言的，正是由于以上总总，教会认为除圣母外，圣若瑟在基督的救恩工程上占有极其重要的地位。这就是教会敬礼大圣若瑟的理由。
