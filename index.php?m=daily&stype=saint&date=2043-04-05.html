复活节庆期第八日(慈悲主日)

　2000年圣年的5月5日,真福教宗圣若望保禄二世发表了《慈悲宽仁》通谕(法令), 在罗马礼仪的弥撒和日课中定下: 复活节后首主日为【救主慈悲主日】。感谢真福教宗, 把救主耶稣基督的慈悲带给了普世教会。

　　　在圣女傅天娜的《日记》里, 耶稣至少十四次, 要教会正式建立一个《慈悲节日》: “我要有一个慈悲庆节。我要妳用彩笔, 绘画这张慈悲圣像, 在复活节后的第一个主日, 隆重地祝圣; 要定该主日为慈悲庆节。”( 日记: 49, 280 ) 耶稣的圣意, 要向我们展示祂救赎人类的复活奥迹, 是与天主救主慈悲的奥迹息息相关。 

     这【救主慈悲主日】不但是一个特别敬礼天主无限慈悲的神圣节日,  而且是天主对世人慷慨普施恩泽的宝贵时刻;  因为在这节日去告解和领圣体的灵魂,  等于得了“第二次洗礼”的所有救赎恩宠,  那就是:“所有罪过及罪罚的完全赦免!  在那天,   我至溫柔的慈悲之最深處敞開了,   并向所有投奔我慈悲泉源的靈魂,   傾注汪洋大海般的恩寵.   在那天,   所有恩宠的水闸都开放了.   让所有灵魂都不必害怕亲近我,   即使他们的罪过红得发紫! ”( 日记: 699)。   获得【救主慈悲主日】的全大赦所需条件为:  完全弃绝罪恶,  在当天或前后的三星期内办告解,  并领圣体圣事,  为教宗意向祈祷,  以及参与【救主慈悲】敬礼和祈祷, 或行其他方式的善工  ( 详情请参阅以下的附录 )。

    主耶稣藉圣女的文笔, 苦口婆心地, 一再劝诫我们:“尽管我为他们忍受了悲慘的苦難,灵魂依旧丧亡。 我现在给他们救赎的最后希望, 那就是: 我的慈悲庆节。如果他们再不崇敬我的慈悲,  他们将永远丧亡! 我慈悲的秘书, 要写下来, 并告诉所有灵魂我的大慈大悲, 因为恐怖的那一天, 我公义的日子近了! ”( 日记： 965 )

     耶稣嘱咐圣女, 要在慈悲节日前,  从受难节星期五开始,  一连九天,  虔诚诵念《慈悲串经》。  祂说:“透过这九日敬礼,  我会赐给灵魂一切所需要的圣宠。”( 日记:796 ) “ 我切愿妳,  在这九日内,  把众灵魂引进我慈悲的泉源中;  好使他们从那里吸取力量,  振作精神,  更得到面对生活中各种艰苦的恩宠,  尤其是临终时所需要的任何圣宠!  每一次,  妳要把一种不同的灵魂,  带领到我的圣心中,  将他们浸润在我慈悲的汪洋大海中。 我将带这些灵魂到我天父的屋子里去!  妳要在今世及来生都这样做!  我决不会拒绝他们所需要的任何恩宠!”( 日记:1209-1229 )  在这 “慈悲九日敬礼”中,  耶稣要我们, 每一天分别引进九类不同的灵魂: 全人类特别是罪人,  神父及修会人士, 热心及忠贞的人, 外教人及未认识基督的人, 散播异端及分裂教会的人, 良善心谦的人及小孩子, 恭敬光荣天主慈悲的人, 炼狱灵魂, 和冷淡灵魂 ( 日记:1209 - 1229 )。

     救主慈悲敬礼的最终目的, 是当我们领受了救主耶稣的慈悲后,  自己也要成为像祂一样慈悲的人。 要尊重关爱邻人如同耶稣,应当怀着如同亲近耶稣一样的爱情,并以恭敬圣体般的恭敬态度服务他们（日记: 285）。                    

“ 我要妳因为爱我的缘故, 实践慈悲的行为。 妳应当时时处处慈悲地对待邻人。不要畏缩不前,或找借口来推却责任。我现在给妳三个慈悲地对待邻人的方法: 第一是具体行动; 第二是透过语言;  第三是借着祈祷。这三种方法,都在不同的程度上,完全概括了慈悲的全部意义。因了这些慈悲的行为,才能毫无疑问地证明妳对我的爱。 透过了这三种行为, 一个灵魂才能光荣和恭敬我的慈悲。 当然,复活节后的第一个星期日就是慈悲庆节,但还应该有慈悲的行为。 我要求人们透过庄严地庆祝慈悲庆节,及崇敬慈悲圣像,来恭敬我的慈悲。 借着这圣像,我会恩赐灵魂无数的圣宠,并时时刻刻提醒世人,我慈悲的要求,因为如果沒有善行, 即使有最坚強的信德也无济于事。”(日记: 742 )

耶稣告诫我们: “ 没有任何灵魂能成义成圣, 除非他以信心投靠我的慈悲, 这便是为什么复活节后的第一个主日, 应当定为慈悲庆节! ”(日记:570)。 愿我们同心同德,为自己,为家庭, 为我们的国家,  为我们的教会, 虔诚祈求慈悲的救主耶稣,对我们大发慈悲。 希望大家每天恳切地诵念《慈悲串经》,默想主的苦难,勤参与弥撒和领圣体,  恭敬圣体和在圣体中的耶稣圣心;在困难的时候,不要忘记说: “耶稣,  我信赖祢! ”祈求痛苦之母,圣母玛利亚,她与耶稣同甘共苦。让我们一起庆祝欢呼: “ 主复活了!  阿肋路亚! ”

 




圣味增爵•斐勒略(St. Vincent Ferrer)

1350年，圣味增爵生于西班牙的法连西亚。幼年受到良好的家庭教育。父母教他虔敬耶稣和圣母玛利亚，并委派他给贫人分发捐款，藉以养成他乐善好施的习惯。他们同时教味增爵每逢瞻礼四和瞻礼七守斋，这项每星期守斋两次的习惯，味增爵终身奉行不辍。

1367年，味增爵入法连西亚的多明我会修院。他在修院内，学业成绩惊人，对神学尤有高深的造诣。未满二十岁，已在加大罗尼最著名的雷利连大学任哲学教授，调往巴塞罗那时，味增爵还是一位六品副祭，但照章已能公开讲道。那时，巴塞罗那发生灾荒，派往别处装运粮食的船，如石沉大海，音讯全无。某日，味增爵在讲道中，向信友预报今晚粮船定将到达码头。院长批评他不应毫无根据发表预言。不料当天晚上，粮船真的到了，居民欣喜若狂，涌往修院向味增爵道贺，恭喜他有预知未来的神能。

不久，味增爵调往故乡传道。由于他学识渊博，口才奇佳，被他劝化的有冷淡的教友也有犹太教徒。其中最著名的，是犹太经师布果（布果后任迦太其那主教）。

味增爵一度患重病，病中吾主耶稣显现给他，圣多明我和圣方济各侍立在耶稣旁边，耶稣叮嘱他追随两位会祖的芳表，讲道训人。味增爵的疾病，当场就痊愈了。他立即启程，遍历法国各城，所到之处常被听道者围得水泄不通，万人空巷。此外，有一批人，常听圣人讲道，听得出了神，舍不得离开他，就跟着他周游各地。这批人最初是散漫而无组织的，后来联合组织一个团体，名叫“味增爵劝化的罪人”，协助圣人开展传教工作。有几位神父也同这批志愿工作人员一起工作，帮助圣人听告解。

1401年至1403年间，味增爵周游西班牙、意大利、瑞士、法兰西等地。他的讲道，以罪恶、死亡、地狱、死后宣判等四端道理为中心。他的讲词十分动人。有时听众惊慌过度，当场晕倒。有时听众感情激动，呜咽于叹息，圣人不得不暂时将讲道中断。

味增爵的兄弟波尼法爵在沙尔德的加多森会修院任院长，味增爵曾多次去那里讲道。修院年鉴记述如下：“天主借味增爵和波尼法爵兄弟二人之手．完成了奇妙的事。听了味增爵讲道而改邪归正的人，从波尼法爵手中领受会衣。”

味增爵在荷兰讲道期内，发显的灵迹不可胜计。相传他也曾横渡海峡，往英格兰、苏格兰、爱尔兰等地，但这一点缺乏证据。

味增爵除本国言语外，还熟悉拉丁和希伯来文，他似乎得到了能通万国方言的神能。因为他所到之处，不同国籍的听众，无论是法国人、德国人、意大利人、或其他国籍人土，都能听懂他每一句话的意义。而且无论听者的人数如何众多，他的讲词，会场每一个角落的听者都能听清楚。

1407年，味增爵重返西班牙。他第一个目的地是格肋纳达。那时格助纳达属莫尔人统治。味增爵在该地讲道期内，一共劝化了八千莫尔人。到了塞维拉和高社华，听众人山人海，教堂无法容纳，只好露天讲道。最后，他回到阔别了15年的故乡法连西亚。

味增爵讲道的效果简直不可思议。他所到之处，赌博亵渎和其他种种罪恶都一扫而空，有仇怨的人立即化敌为友握手言和。有一天，味增爵在撒拉孟格广场作露天讲道，所讲的题目是他平日最爱讲的“万民四末”。他自称为“报导公审判的大使”。有几个听者听见得这话说得太过分，表示不满。味增爵抬头一看，旁边恰巧有人抬着一个女子的尸体走过，往坟场去埋葬。味增爵说道：“那个亡者可以为我作证。”他就问那尸体：“我刚才那句话究竟说得对不对？”那尸体真的坐起来开口说话，证明味增爵确是报导公审判的天使，说完就又倒下去。当然，味增爵自称为“报导公审判的天使”，并非指自己真的是一位“天使”，他的意思是说：我常在讲公审判的道理，就如报导公审判的天使一样。所以这句话是一种借喻。

味增爵晚年，远居法国。他已是一位龙钟老人，走路需人搀扶，可是他一上讲道坛就声音洪亮，精神抖擞，与壮年时代相比毫无逊色。

1419年，在诺脱讲完了道回到万城后，突患重病，于苦难主日内瞻礼四与世长辞，享年70岁。1455年荣列圣品。

味增爵一生最杰出的德行是谦德。自古以来的大圣人都是如此，他们越接近天主，越认识自己如何卑不足道。




真福儒利亚纳(童贞 Bd. Juliana of Mount Cornillon)

每年圣神降临节后第一主日内瞻礼五那一天，圣教会庆祝耶稣圣体圣血节。第一个提倡这节日的不是别人，就是本文所述的福女儒利亚纳。

1192年，儒利亚纳在列日诞生。五岁时父母去世。儒利亚纳和她胞妹依搦斯二人由奥斯定会修女抚养。那修女院附设病院一座，收容麻疯病人。院长把这二个女孩放在邻地农场，托一位修女照管，以免传染麻疯病。

依搦斯不久就夭亡了。儒利亚纳读书识字，她特别恭敬圣体圣事，暇时爱好阅读圣奥斯定、圣伯尔纳铎和其他教父的著作。

福女自16岁起，日日夜夜看见一个圆形的明月呈现在她眼前，月亮中间呈现出一条黑带。有一次吾主耶稣显现给她，说这圆形的明月便是圣教一年四季周期性的各种节日，那条黑带指的是，全年瞻礼中还缺少一个瞻礼，需要恭敬圣体宝血的庆节。儒利亚纳把神示的情形谨记在心，可是她人微言轻不敢有所表示。

岁月如流，儒利亚纳正式发愿成了一位修女。1225年，当选为女修院院长后，就开始把这团久藏在心头的秘密向一位隐修女（福女伊华）和另一位很有圣德的修女透露。接着她把这件事向一位博学的神父提出，问他在神学的立场是否可以设置一个恭敬圣体圣血的节日。

教会当局对这问题开始注意，雅各伯庞德隆（后荣任教宗，即教宗乌尔巴诺四世）、多明我会会长、巴黎大学校长和其他很有权威的神学家，都表示在神学方面绝无反对设置恭敬圣体圣血的节日的理由。可是神圣的计划，有时免不了受到目光短浅之徒的反对和阻挠，列日的信友强迫儒利亚纳离境。主教下令调查，将圣体圣血节列入列日教区的庆节表，并召回儒利亚纳。在主教去世后，儒利亚纳又被迫离境，悄悄带领三位修女流亡各地求乞度日。后来撒西尼女修院院长见义勇为，收容她们食宿。儒利亚纳在这个世界上似乎应当受种种苦难折磨，现世幸福她根本无缘享受，这一点她自己知道的很清楚。儒利亚纳好不容易在撒西尼获得栖身之所，不久战事爆发，撒西尼修院被毁，她同院长逃到福萨修院。暮年贫病交迫，生活凄苦。1258年4月逝世。死时仅有院长和另一位修女在旁。

儒利亚纳撒播的种子萌芽茁长，开花结果。她的知友福女伊华，继续她的任务，全力提倡耶稣圣体圣血节。那时拥推圣体圣血节的雅各伯庞德隆已荣登教宗宝座，伊华就托列日主教上书教宗。教宗颁诏，钦定耶稣圣体圣血节，并令伟大的神哲学家圣多玛斯编写耶稣圣体圣血节的经文。为了嘉奖伊华提倡耶稣圣体圣血节的功劳，教宗特颁赐她“钦定耶稣圣体圣血节”的诏文和多玛斯的耶稣圣体圣血节的经文各一份。儒利亚纳生前，虽未耳闻目睹耶稣圣体圣血节在教会内普遍举行，可是当她的在天之灵看到数百年来，普世教会热烈奉行耶稣圣体圣血节如火如荼的盛况，一定觉得很愉快的。

儒利亚纳于1869年荣列真福品。
