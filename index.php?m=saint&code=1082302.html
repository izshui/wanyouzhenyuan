圣斐理伯•贝尼谛
St. Philip Benizi
 8月23日

圣斐理•贝尼谛于1233年8月15日圣母升天瞻礼出生，原籍意大利佛罗伦萨，自幼热心敬主，先后在巴黎和帕雕亚攻读医学。年19岁，考取医学博士、哲学博士学位。

斐理伯在佛罗伦萨执行医生业务，同时潜心研究圣经和教父的著作。他常在斐沙圣堂的苦像前，和圣母忠仆会(即撒未德会)小堂的圣母领报像前祈祷。

那时候，撒未德会成立仅十四年。会士分居小室，严操苦行。公元1254年的复活瞻礼，斐理伯在斐沙圣堂祈祷，似乎听到苦像上的耶稣对他道：“你到山上的圣母忠仆会去，这些修士是我母亲的忠仆。”斐理伯听了，默想片刻，到撒未德会小堂望弥撒。这一天弥撒经的“书信”是宗徒大事录斐理伯为爱西屋比亚太监行洗礼的一段。弥撒经的书信有一段话：“圣神对斐理伯说，你靠近这辆车子……”斐理伯•贝尼谛的名字也是斐理伯，他以为这句话很巧，能够适用在自己身上。天主似乎叫他入撒未德会，置身在圣母的照顾之下。

斐理伯在神视中，看见自己在一个旷野的地方，四面都是毒蛇，他很害怕，不知怎样才能逃走，突然圣母坐着马车走近。斐理伯恍然大悟，天主要他入圣母忠仆会修道，他就到西诺利山的圣母忠仆会修院，请求圣朋费利收录为辅理修士。他很谦逊，不肯透露自己的才学。他说：“我愿意做圣母忠仆的仆人。”

斐理伯在修院里当园丁，出外募捐，并担任各项辛苦的杂役；他一面工作，一面默想祈祷。

1258年斐理伯被派往栖亚纳的圣母忠仆会修院。有一天，修士们讨论一个很难的哲学问题，斐理伯一时不小心，发表了意见。人们听了，才知道他是有学问的人。会长考查了斐理伯的才学，祝圣他为司铎。

1262年，斐理伯任西亚纳修院的初学导师，兼任总会长的助理员。

1267年，圣母忠仆会举行全体大会，一致推选斐理伯为总会长。他接任后，巡视意大利北部各修院。亚理沙的修院缺乏粮食，斐理伯到了，劝大家全心依靠天主。他在圣母像前祈祷，祈祷完毕，有人在修院门外敲门，修士开了门，有两大盘面包放在地上。

教宗格肋孟四世去世，众人有意推选斐理伯继任。斐理伯得了这消息，逃到深山里躲藏，待新教宗选出，方才重返修院。他在山洞里住了3个月，远离尘嚣，一天到晚，默想祈祷，吃的是野菜，饮的是泉水，后来这山泉称为“斐理伯泉”。

1274年，斐理伯应教宗真福额我略十世之召，出席里昂第二次大公会议。

斐理伯口才奇佳，顽固的罪人，被他劝化的，不计其数，意大利各地的政治派系，发生冲突，斐理伯到处作鲁仲连，劝他们息争议和。福利的乱党殴打他，他一点不动怒，他的忍耐精神感动了众人。乱党的领袖彼利格尼动手打过圣人，后来跪在他面前，向他谢罪。1283年，彼利格尼弃俗修道，入圣母忠仆会修道，由斐理伯之手领受会衣，死后荣列圣品(彼利格尼于1726年由教宗本笃十三世列入圣品)。

斐理伯一手培植的著名热心教友，除了圣彼利格尼以外，还有真福若望•佛兰福、真福若亚敬•毕古洛弥、真福安德肋•杜田、真福热罗尼莫、真福文都辣•毕斯多(毕斯多性情暴戾，作恶多端，听了斐理伯的讲道，悔罪改过)，真福乌巴、真福方济各•巴弟西。

1284年，圣亚理西请斐理伯指导他侄女圣女儒利亚纳的灵修生活。圣女儒利亚纳获得了斐理伯的指示，全力发展圣母忠仆会第三会。

斐理伯派遣了圣母忠仆会的首批传教士往东方，一部分圣母忠仆会会士抵达多塔，为主流血致命。

斐理伯任圣母忠仆会会长达18年之久，忠勤尽职，荣主救灵，对教会、对修会都有莫大的贡献。

斐理伯暮年体弱多病，自知在世岁月无多，于1285年往比鲁日朝觐新教宗何诺留四世，在佛罗伦萨召开圣母忠仆会全体大会，将会长职务移交洛太林古神父(洛太林古与斐理伯合作多年，襄助斐理伯处理会务，功绩卓著)，他在会场一再敦劝会士；“你们应当彼此相爱，你们应当彼此相爱。”

会议结束，斐理伯往多田修院。多田教友热烈欢迎斐理伯到小堂圣母祭台，跪地祈祷：“这是我永远安息的处所。”圣母升天瞻礼，斐理伯讲道，描述至圣圣母的光荣，听者无不动容；同日下午三时，突患病重，他召集同院修士，敦切嘱咐：“你们应当彼此相爱，你们应当彼此相敬，你们应当彼此容忍。”

7日后，斐理伯病重，他叫旁人将他的“书”拿来，别人起初不懂，后来才知道，他说的“书”就是他的苦像。他手捧着苦像，举目瞻视，在晚上三钟经的时候，安逝主怀。

斐理伯•贝尼谛于1671年荣列圣品。1694年起，普世教会均在本日举行瞻礼，纪念这位伟大的圣人。
