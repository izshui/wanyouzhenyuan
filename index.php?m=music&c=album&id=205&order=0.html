                                                        #人物传记
                                                    

                        

                            

                                
贾发束是十九世纪意大利其中一位最重要的伦理神学家和圣职人员神师。贾发束神父探访医院、慈善机构、茅舍、邻近村庄和市镇，所有人也受惠于他的爱德和热诚。他尤其关心囚犯，“可以说，他在探访监狱时，就像如鱼得水，罪犯都是他所爱的儿子。他在帮助囚犯时，内心找到宣发感情的渠道”。（BM II, 48）。他特别关心死囚，曾陪伴六十八个死囚上绞刑架，直至他们给绞死，因此有“绞刑架的神父”之美誉。瓜拉神父经常对司铎学员说：“去找若瑟吧”。贾发束神父带鲍思高神父接触囚犯。他常在休憩时间，讲述他在医院、监狱和其他地方遇到的事。有个行刑者告诉贾发束神父：“有你在旁，他们不再视死亡为死亡，而视之为安慰、喜乐和赏报”。
